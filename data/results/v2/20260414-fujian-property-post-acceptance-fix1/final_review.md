## 风险点1：评分标准中设置特定品牌倾向性条款

- 问题定性：高风险
- 审查类型：评分因素不相关/技术参数倾向性
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 加分项（F4×A4）
- 原文摘录：“设备配置如果有华为小米等相关国产品牌优先。”
- 风险判断：
  - 该条款在“加分项”中直接点名“华为”、“小米”等特定品牌，属于典型的以特定品牌作为加分条件。
  - 物业管理服务采购中，设备配置通常作为服务的一部分，但直接指定品牌作为加分项，限制了其他同等质量国产品牌供应商的公平竞争机会。
  - 违反了政府采购中不得以特定品牌、型号作为评审因素的规定，具有明显的指向性和排他性。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；……（八）以其他不合理条件限制或者排斥潜在供应商。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 删除“华为小米等相关国产品牌优先”的表述。
  - 若需鼓励国产品牌，应表述为“设备配置采用国产品牌（需提供相关证明材料）”，且不得指定具体品牌。
  - 若需对设备性能进行评分，应设定具体的技术参数指标（如功能、性能参数等），而非品牌名称。

## 风险点2：技术参数中人员配置要求存在性别、年龄等歧视性条款

- 问题定性：中风险
- 审查类型：合规性审查
- 原文位置：证据 2，第 995 - 1054 行
- 原文摘录：工程技术人员（含水电工、维修/电梯工、弱电工、运行工）25-55 周岁... 环境领班 25-45 周岁... 实验室洗涤人员 20-55 周岁... 动物房饲养员 20-55 周岁... 其他保洁人员 20-55 周岁
- 风险判断：
  - 招标文件对多个岗位设定了具体的年龄上限（如 55 周岁、45 周岁），这可能构成对大龄劳动者的就业歧视。
  - 根据《就业促进法》及相关政府采购政策，除国家规定的特殊工种外，一般不应设置与履行合同能力无直接关联的年龄限制。
  - 物业管理服务中，部分岗位（如保洁、维修）的体力要求虽存在，但直接设定年龄上限缺乏充分必要性论证，可能排斥了具备相应能力的潜在投标人。
- 法律/政策依据：
  - 《中华人民共和国就业促进法》第二十七条：用人单位招用人员，不得以是传染病病原携带者为由拒绝录用。但是，经医学鉴定传染病病原携带者在治愈前或者排除传染嫌疑前，不得从事法律、行政法规和国务院卫生行政部门规定禁止从事的易使传染病扩散的工作。（注：虽未直接提及年龄，但结合反就业歧视原则）
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；（八）以其他不合理条件限制或者排斥潜在供应商。
  - 《关于在政府采购活动中查询及使用信用记录有关问题的通知》及各地关于清理政府采购中不合理限制的规定，通常禁止设置与履行合同无关的年龄、性别等限制。
- 整改建议：
  - 建议删除对拟派人员的年龄限制要求，改为要求人员身体健康、具备相应岗位所需的技能和经验。
  - 如确因岗位特殊性（如高空作业等法定年龄限制）需要，应提供明确的法律法规依据，并仅针对特定岗位设定，而非全员设定。

## 风险点3：商务条款中关于“无犯罪证明”的提交时限及无效投标处理存在法律风险

- 问题定性：中风险
- 审查类型：商务条款失衡/不合理条件
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （二）物业管理团队人员配备情况及要求 -> 2)
- 原文摘录：“投标人须单独提供承诺函（格式自拟），承诺在合同签订后的三个月内提供无犯罪证明，未提供的按无效投标处理。”
- 风险判断：
  - 招标文件要求投标人在“合同签订后的三个月内”提供无犯罪证明，否则“按无效投标处理”。
  - “无效投标”通常指在开标、评标阶段因不符合招标文件实质性要求而被否决投标的情形。将“合同签订后”的履约行为（提供无犯罪证明）作为“无效投标”的处理依据，逻辑上存在矛盾。投标阶段无法预知合同签订后的情况，且“无效投标”的认定应在评标阶段完成。
  - 正确的做法应是将“提供无犯罪证明”作为中标后的合同履约条件或验收条件，若未提供，采购人有权解除合同或追究违约责任，而非追溯认定为“无效投标”。
  - 此外，要求所有拟派人员（包括普通保洁、保安等）在中标后三个月内提供无犯罪证明，可能涉及个人隐私保护及实际操作难度，需评估其必要性和合理性。
- 法律/政策依据：
  - 《中华人民共和国政府采购法》第五十条：政府采购合同的双方当事人不得擅自变更、中止或者终止合同。
  - 《中华人民共和国政府采购法实施条例》第四十九条：中标或者成交供应商拒绝与采购人签订合同的，采购人可以按照评审报告推荐的中标或者成交候选人名单排序，确定下一候选人为中标或者成交供应商，也可以重新开展政府采购活动。
- 整改建议：
  - 修改条款表述，将“按无效投标处理”改为“视为中标人违约，采购人有权解除合同，不予退还履约保证金，并追究相应法律责任”。
  - 明确“无犯罪证明”的提供范围（如仅限关键岗位人员：项目经理、安防主管、消控员等），避免对所有普通员工提出过高要求。
  - 确保合同条款与招标文件的一致性，避免在评标阶段设置无法在投标时验证的“无效投标”条件。

## 风险点4：评分标准中设置与履约能力关联度不高的企业荣誉及特定荣誉加分

- 问题定性：中风险
- 审查类型：评分因素相关性审查
- 原文位置：证据6，第722-772行，商务项第1点
- 原文摘录：1.企业荣誉 3.00 投标人2020年1月1日以来（以获奖时间为准）获得过地市级（含）以上人民政府或行政机关颁发的与物业服务或企业诚信经营有关的荣誉或表彰的，每提供1份得1分，满分3分。
- 风险判断：
  - 根据《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条，评审因素应当细化和量化，且与相应的商务条件和采购需求对应。
  - 企业荣誉（如诚信表彰、优秀企业等）属于企业过往的综合性荣誉，与本项目具体的物业管理服务质量、履约能力无直接因果关系。
  - 将此类荣誉作为评分项，可能导致评分结果受企业历史背景影响，而非基于本项目实际服务能力，存在排斥潜在投标人或倾向特定企业的风险。
  - 建议删除该评分项，或将其调整为与项目直接相关的服务承诺、质量保障体系等实质性内容。
  - “企业荣誉”项中“与物业服务或企业诚信经营有关”的表述过于宽泛，缺乏量化标准，且荣誉的获取往往与企业的历史积累、公关能力相关，与本项目具体的履约能力（如人员素质、管理方案、应急响应等）无直接必然联系。
  - “项目荣誉”中要求获得“节约型公共机构示范单位”、“城区生活垃圾分类工作先进单位”等特定荣誉，这些荣誉通常是针对业主单位（采购人）或特定项目的综合评定，而非物业服务企业本身的履约能力证明。将业主单位的荣誉或特定项目的荣誉作为对服务企业的评分项，逻辑上存在偏差，且可能限制未获得此类特定荣誉但具备同等服务能力的中小企业参与竞争。
  - 此类评分因素容易导致“唯荣誉论”，偏离了“择优”的采购目标，可能构成以不合理的条件限制竞争。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条
  - 《政府采购需求管理办法》（财库〔2021〕22号）第二十一条
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。商务条件和采购需求指标有区间规定的，评审因素应当量化到相应区间，并设置各区间对应的不同分值。
  - 《政府采购需求管理办法》（财库〔2021〕22号）第二十一条：采购需求客观、明确，且资金及交易条件允许市场供应的，应当通过综合评分法进行评审。评审因素应当与采购标的的履约能力直接相关。
- 整改建议：
  - 建议删除‘企业荣誉’评分项。
  - 若需考察企业信誉，可改为考察‘近三年内无重大违法记录’（作为资格条件）或‘提供针对本项目的质量承诺及违约赔偿方案’（作为技术/商务评分项），确保评分因素与履约能力直接相关。
  - 删除“企业荣誉”中关于“诚信经营”等宽泛荣誉的评分项，或将其改为与本项目直接相关的“服务质量管理体系认证”（如ISO9001等）且需与物业管理相关。
  - 删除“项目荣誉”中针对特定荣誉称号（如“节约型公共机构示范单位”等）的评分项。
  - 将评分重点转向与项目履约直接相关的因素，如：类似项目业绩（合同金额、服务内容）、项目团队人员资质、服务方案的可操作性、应急预案的完善程度等。

## 风险点5：评分标准中“信息化软件服务能力”要求著作权人为投标人，可能限制竞争

- 问题定性：中风险
- 审查类型：评分因素不相关/限制竞争
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 商务项（F3×A3）-> 6.信息化软件服务能力
- 原文摘录：“投标人拥有与物业服务相关信息化管理软件或系统，并承诺在服务期内该软件或系统会投入本项目使用的，每提供1项得1分，满分3分。投标人须提供承诺函和著作权人为投标人的《计算机软件著作权登记证书》……"
- 风险判断：
  - 要求“著作权人为投标人”意味着投标人必须自行开发软件，这排除了通过购买、租赁或授权方式使用成熟商业软件（SaaS模式）的供应商。
  - 对于物业管理服务而言，使用成熟的第三方软件往往比自行开发更稳定、更高效。强制要求“著作权人为投标人”不仅增加了中小企业的成本负担（开发软件成本高），而且与项目履约能力（能否提供优质服务）无直接关联。
  - 该条款可能构成以不合理的条件限制竞争，排斥了拥有成熟软件解决方案但非自研的供应商。
  - 正文明确要求著作权人为投标人，属于以特定权利主体限定评分条件，具有明显限制竞争效果。
  - 该条件将软件使用能力与软件著作权归属强绑定，排斥合法授权使用同类软件的供应商。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
  - 《政府采购促进中小企业发展管理办法》（财库〔2020〕46号）：不得以不合理的条件限制中小企业参与政府采购活动。
  - 评审因素应与履约能力直接相关，不得设置不合理限制竞争条件。
- 整改建议：
  - 修改评分标准，允许投标人提供“拥有自主知识产权”或“合法授权使用的”信息化管理软件。
  - 将评分重点放在软件的功能、稳定性、与本项目需求的匹配度上，而非著作权归属。
  - 建议表述为：“投标人拥有或合法授权使用的与物业服务相关信息化管理软件或系统……需提供软件著作权证书或合法授权证明文件”。
  - 删除“著作权人为投标人”的限制，改为核验软件实际可用性或合法授权证明。

## 风险点6：技术参数中关于“实验服清洗服务”的设备品牌指定风险

- 问题定性：低风险
- 审查类型：技术参数倾向性
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> H.其他服务 -> 2.实验服清洗服务
- 原文摘录：“由中标人提供8台全新滚筒洗衣机（容积大于等于10公斤、洗烘一体，如：美的、海尔、小天鹅同品质品牌）供检验科所使用……"
- 风险判断：
  - 条款中使用了“如：美的、海尔、小天鹅同品质品牌”的表述。虽然使用了“如”和“同品质品牌”，但在实际评审中，评委可能倾向于认为只有这三个品牌或与其完全同等的品牌才符合要求，从而产生倾向性。
  - 对于服务类项目中的设备配置，应侧重于设备的技术参数（如容积、功能、能效等级等），而非列举具体品牌。
  - 尽管风险相对较低（因为有“同品质品牌”的兜底），但仍建议优化表述以避免歧义。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品。
- 整改建议：
  - 删除“如：美的、海尔、小天鹅”等具体品牌名称。
  - 修改为：“由中标人提供8台全新滚筒洗衣机（容积大于等于10公斤、洗烘一体，品牌不限，但需提供产品合格证及技术参数证明，确保品质满足检验科所使用要求）”。

（1）首先由主持人宣布开标会须知，然后由投标人代表对电子投标文件的加密情况进行检查，经确认无误后，由工作人员对参加现场开标会投标人的电子投标文件进行解密。通过远程参与开标流程的投标人须在系统远程解密开启后，在代理机构规定时间内使用CA数字证书进行电子投标文件的解密操作，逾期未解密的视为放弃投标。
（2）唱标时，唱标人将依次宣布“投标人名称”、“各投标人关于电子投标文件补充、修改或撤回的书面通知（若有）”、“各投标人的投标报价”和招标文件规定的需要宣布的其他内容（包括但不限于：开标一览表中的内容、唱标人认为需要宣布的内容等）。
（3）唱标结束后，参加现场开标会的投标人代表应对开标记录进行签字确认，通过远程参与开标流程的投标人须在系统远程签章开启后，在系统规定时间内对开标结果进行签章确认。
（4）投标人代表对开标过程和开标记录有疑义，以及认为采购人（采购代理机构）相关工作人员有需要回避情形的，应当场或通过系统提出询问或回避申请。投标人代表未按规定提出疑义又拒绝对开标记录签字或通过系统远程签章确认的，视为投标人对开标过程和开标记录予以认可。
（5）若投标人未到开标现场参加开标会，也未通过远程参加开标会的，视同认可开标结果。
