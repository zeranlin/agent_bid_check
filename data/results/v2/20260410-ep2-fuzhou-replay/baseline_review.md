## 风险点1：技术参数引用错误标准及标准号存疑

- 问题定性：高风险
- 审查类型：技术参数倾向性/合规性
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> 序号1：组合柜 -> 第8点；序号2：学生宿舍座椅 -> 第8点；序号3：楼梯踏板 -> 第4点
- 原文摘录：- 组合柜第8点：“家具中甲醛、苯、二甲苯和TVOC的释放量以及家具中可迁移有害元素限量需要符合GB 28481-2012《家具中有害物质限量》标准以及GB 48001-2026《汽车车门把手安全技术要求》标准中的要求。”
- 座椅第8点：“家具中甲醛、苯、二甲苯和TVOC的释放量，家具中可迁移有害元素限量，以及领苯二甲酸脂、多环芳烃限量需要符合GB18584-2024《家具中有的害物质限量测量》标准中的要求。”
- 楼梯踏板第4点：“家具中可迁移有害元素限量，以及领苯二甲酸脂、多环芳烃限量需要符合GB18584-2024《家具中有害物质限量》标准中的要求并提供福建省检测机构出具的检测报告。”
- 风险判断：
  - **标准引用严重错误**：组合柜要求中引用了"GB 48001-2026《汽车车门把手安全技术要求》”，该标准名称与家具采购标的完全无关，且年份为2026年（未来时间），属于明显的编造或错误引用，导致技术参数无法执行，构成排斥潜在供应商。
  - **标准号及名称存疑**：座椅和楼梯踏板引用的"GB18584-2024"及《家具中有的害物质限量测量》（错别字“有的害”）存疑。现行有效标准通常为GB 18584-2001《室内装饰装修材料 木家具中有害物质限量》或GB 28481-2012《家具中有害物质限量》。2024版标准是否存在需核实，且标准名称存在明显文字错误，影响评审的客观性和准确性。
  - **地域限制风险**：楼梯踏板要求“提供福建省检测机构出具的检测报告”，限制了外地检测机构出具的报告，涉嫌以不合理的条件限制或者排斥潜在供应商。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；（八）以其他不合理条件限制或者排斥潜在供应商。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 立即核实并修正所有引用的国家标准号及标准名称，删除与家具无关的“汽车车门把手”标准。
  - 确认GB 18584的最新有效版本及正确名称，修正错别字。
  - 删除“福建省检测机构”的地域限制，改为“具有CMA或CNAS资质的检测机构出具的检测报告”。

## 风险点2：样品制作要求具有排他性及泄露投标人信息风险

- 问题定性：高风险
- 审查类型：技术参数倾向性/限制竞争
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （二）样品要求 -> 第3点、第4点
- 原文摘录：- 第3点：“投标人应采用可去除的安全材料隐去投标人名称及其他可能泄露投标人信息的标志、符号等；同时，在投标文件中，也不得出现样品图样，标识等可能泄露投标人样品的任何信息，否则按无效投标处理。”
- 第4点：“投标人必须将以上样品成品提前组装好，并于开标当日投标截止时间前一次性送达投标现场，按要求摆放好样品。必要时评标现场评委可对所有样品进行测试，若出现复核情况时，评标委员会对样品不作再次评审，样品评审结果不作更改。”
- 风险判断：
  - **样品评审方式风险**：要求“开标当日投标截止时间前一次性送达投标现场”，对于异地供应商而言，物流时间难以控制，若因物流原因导致样品迟到，直接导致无效投标，增加了供应商的履约成本和风险，可能构成以不合理的条件限制竞争。
  - **信息泄露与盲评矛盾**：虽然要求隐去信息，但要求“提前组装好”且“一次性送达”，若样品本身具有独特的工艺特征（如特定的注塑模具痕迹、特殊的连接结构），评委极易通过样品特征识别出特定供应商，导致“盲评”流于形式，实质上指向特定供应商。
  - **复核机制缺失**：规定“若出现复核情况时，评标委员会对样品不作再次评审”，剥夺了供应商对样品评审结果提出异议和复核的权利，程序不公。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。
  - 《政府采购法实施条例》第二十条：不得以不合理的条件对供应商实行差别待遇或者歧视待遇。
- 整改建议：
  - 建议改为“样品在评标现场由评委随机抽取评审”或允许供应商在开标前将样品送至指定地点由代理机构统一保管，避免物流风险。
  - 明确样品评审的复核机制，允许在特定条件下进行复核。
  - 确保样品评审标准客观量化，减少主观判断空间。

## 风险点3：商务条款中采购人单方变更权过大且结算方式不明

- 问题定性：中风险
- 审查类型：商务条款失衡
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 其他商务要求 -> 8.3、13.3
- 原文摘录：- 8.3：“签订合同后，采购单位有权根据实际需要对本次采购文件中未明确或未规定的事项作出规定或细微调整，及对各种货物的款式进行细微调整（允许偏离±5%），最终以采购人实际为准，中标人的最终报价不变，如遇采购人特殊需求特殊制作与招标尺寸有较大差异的货物，偏离超过5%，结算单价按调整后面积比例相应换算。”
- 13.3：“①本项目中标后，采购人有权对家具的颜色、款式、尺寸等作出细微调整，中标价不变，中标人不得因此增加任何费用或持有任何异议。②...采购人根据现场的实际情况，有权对所有货物的数量、尺寸和款式进行增加、减少或调整，最终以采购人实际的安装数量为准，变化部分的单价与投标报价对应的单价一致..."
- 风险判断：
  - **合同变更权失衡**：条款赋予采购人单方面调整款式、尺寸、颜色的权利，且规定“中标价不变”、“不得增加任何费用”。家具定制涉及材料成本、工艺难度，若调整导致成本增加（如颜色变更需重新调色、尺寸变更导致材料浪费），强制中标人承担所有风险，显失公平。
  - **结算依据模糊**：“按调整后面积比例相应换算”缺乏明确的计算公式和单价调整机制，容易在履约过程中产生争议。
  - **固定单价风险**：虽然名为固定单价，但结合“细微调整不增加费用”的条款，实际上将市场风险和变更风险全部转嫁给供应商，可能导致供应商在投标时虚高报价以覆盖风险，或中标后偷工减料。
- 法律/政策依据：
  - 《中华人民共和国民法典》第六条：民事主体从事民事活动，应当遵循公平原则，合理确定各方的权利和义务。
  - 《政府采购法》第四十九条：政府采购合同履行中，采购人需追加与合同标的相同的货物、工程或者服务的，在不改变合同其他条款的前提下，可以与供应商协商签订补充合同，但所有补充合同的采购金额不得超过原合同采购金额的百分之十。
- 整改建议：
  - 修改条款，明确“细微调整”的定义及范围。
  - 约定若调整导致成本增加，应通过协商签订补充协议调整价格，或明确合理的成本补偿机制。
  - 细化“面积比例换算”的具体计算公式，确保结算透明。

## 风险点4：验收条款中“驻厂检查”及“终止合同”条件过于严苛

- 问题定性：中风险
- 审查类型：商务条款失衡/验收条款
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 其他商务要求 -> 10.4
- 原文摘录：- 10.4：“投标人须承诺：若中标，在生产本次采购家具的过程中，采购人可对产品从进料、开胚、酸洗磷化、喷涂、成型等一系列生产程序进行驻厂检查、监督；如果采购人在检查监督过程中发现中标人未按照相关工艺要求进行生产，采购人有权终止合同，并按照国家相关法律法规的规定提请有关政府监管部门对其进行处理。”
- 风险判断：
  - **驻厂检查权利过大**：要求采购人可介入“进料、开胚、酸洗磷化、喷涂、成型”等所有生产环节进行“驻厂检查”，对于供应商的生产管理、商业秘密保护构成极大干扰，且增加了供应商的接待成本。
  - **终止合同条件过于严苛**：仅因“发现未按照相关工艺要求生产”即可直接“终止合同”，未区分一般瑕疵与重大违约，也未给予整改机会。若因轻微工艺偏差（如非关键部位）即终止合同，对供应商过于严厉，可能导致供应商不敢投标或产生纠纷。
- 法律/政策依据：
  - 《政府采购法》第五十条：政府采购合同的双方当事人不得擅自变更、中止或者终止合同。
  - 《民法典》第五百六十三条：当事人一方迟延履行债务或者有其他违约行为致使不能实现合同目的，当事人可以解除合同。
- 整改建议：
  - 将“驻厂检查”改为“不定期抽查”或“关键节点验收”，明确检查范围和频率。
  - 修改终止合同条款，增加“经采购人提出整改要求后，中标人未在规定期限内整改或整改后仍不符合要求”的前置条件，区分一般违约与根本违约。

## 风险点5：技术参数中部分指标过于具体，存在指向特定产品嫌疑

- 问题定性：中风险
- 审查类型：技术参数倾向性
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> 序号1：组合柜 -> 第1点、第2点、第5点；序号2：学生宿舍座椅 -> 第3点、第4点
- 原文摘录：- 组合柜第1点：“档条高度130mm（±5mm）...插座中具备2个五孔插座和1个USB插座...插座需要内嵌在档板中...挂钩锁样式见附图。”
- 组合柜第2点：“防撞角规格为40*40mm（±5mm），厚度≥20mm...外露面表面具有≥3条弧形内凹槽。”
- 组合柜第5点：“拉手主体上下两头圆弧设计，中间位置内凹设计...规格尺寸为123*28*22mm（±5mm）...拉手盖板上下两头圆弧形，规格为120*25*22mm（±5mm）。”
- 座椅第3点：“座板中间最宽处宽度435mm（±5mm）...连接部件宽度为150mm（±5mm）...中间位置高度38mm（±5mm）。”
- 风险判断：
  - **参数过于具体**：拉手、防撞角、座板连接件等部件的尺寸精确到毫米（如123*28*22mm），且对形状（如“中间位置内凹”、“弧形内凹槽”）有详细描述。这些非核心功能参数过于具体，极有可能是照搬某一家特定供应商的产品参数，导致其他供应商难以响应或需专门开模，构成指向特定产品。
  - **附图依赖**：多处提到“样式见附图”、“外形以及规格尺寸见附图”，若附图中的设计具有独特性，将进一步强化排他性。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
  - 《政府采购法实施条例》第二十条：不得以不合理的条件对供应商实行差别待遇或者歧视待遇。
- 整改建议：
  - 将精确尺寸改为范围值（如“拉手尺寸约为120mm左右”或“满足人体工学要求”），或仅规定功能要求（如“拉手应便于抓握”）。
  - 删除对非关键部件形状、凹槽数量等细节的强制性描述，改为“样式美观、符合安全要求”等定性描述。
  - 确保附图仅为参考，不作为唯一验收标准。

## 风险点6：履约保证金退还时限与法律规定不一致

- 问题定性：低风险
- 审查类型：商务条款合规性
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 6.★合同支付方式 -> 7.★履约保证金
- 原文摘录：- “该履约保证金将在验收合格后且中标人无违约的前提下10个工作日内无息退还（转账方式提交的）或解除担保（保函形式提交的）。”
- 风险判断：
  - **时限合规性**：虽然10个工作日在合理范围内，但需确认是否与《政府采购法实施条例》及当地财政部门关于优化营商环境的最新要求（如“验收合格后5个工作日内”）保持一致。若当地有更严格规定，此条款可能滞后。
  - **无息退还**：对于转账方式，无息退还是常规做法，但需确认是否涉及资金占用利息的争议。
- 法律/政策依据：
  - 《政府采购法实施条例》第四十八条：采购人与中标、成交供应商应当在中标、成交通知书发出之日起三十日内，按照采购文件确定的事项签订政府采购合同。
  - 各地优化营商环境条例（如福建省相关规定）可能对保证金退还有更短时限要求。
- 整改建议：
  - 核实福建省及福州市关于政府采购履约保证金退还的最新时限要求，如有更短时限（如5个工作日），建议修改以符合最新政策。

## 风险点7：评分办法未体现技术优劣，仅采用最低评标价法

- 问题定性：中风险
- 审查类型：评标方法
- 原文位置：第四章 资格审查与评标 -> 二、评标 -> 7、评标方法和标准
- 原文摘录：- “7.1评标方法：采购包1：最低评标价法”
- “7.2评标标准：采购包1：最低评标价法。投标文件满足招标文件全部实质性要求，且投标报价最低的投标人为中标候选人。”
- 风险判断：
  - **方法适用性**：本项目为定制家具，涉及设计、材质、工艺、环保等多重因素，且技术参数中包含了大量细节要求。采用“最低评标价法”可能导致供应商为中标而恶意低价竞争，进而牺牲产品质量、环保标准或售后服务，难以保证“物有所值”。
  - **风险转移**：虽然招标文件设置了严格的实质性条款（★号条款），但在最低评标价法下，只要满足★号条款，价格最低者即中标，无法通过评分机制筛选出技术更优、服务更好的供应商。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十四条：政府采购货物和服务项目，采用最低评标价法的，评标结果按投标报价由低到高顺序排列。但该方法适用于“技术、服务等标准统一，采购人没有特殊要求，或者技术、服务等标准统一，采购人没有特殊要求”的项目。
  - 本项目为定制家具，技术和服务标准难以完全统一，建议采用综合评分法。
- 整改建议：
  - 建议将评标方法改为“综合评分法”，设置技术、商务、价格等评分项，重点考察产品质量、环保性能、售后服务方案等，避免唯价格论。

## 综合判断

- 高风险问题：
  - 风险点1：技术参数引用错误标准及标准号存疑（标准错误、地域限制）。
  - 风险点2：样品制作要求具有排他性及泄露投标人信息风险（物流风险、盲评失效）。
