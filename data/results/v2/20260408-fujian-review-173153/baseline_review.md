# 招标文件合规审查结果

审查对象：`福建省食品药品质量检验研究院2023年物业管理服务采购`

说明：
- 本审查基于你提供的招标文件文本进行。
- 对于存在事实基础不足、需要采购人补充论证材料才能最终定性的事项，明确标注“需人工复核”。

## 风险点1：评分标准中设置特定品牌倾向性条款

- 问题定性：高风险
- 审查类型：评分因素不相关/技术参数倾向性
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 加分项（F4×A4）
- 原文摘录：“设备配置如果有华为小米等相关国产品牌优先。”
- 风险判断：
  - 该条款在“加分项”中直接点名“华为”、“小米”等特定品牌，属于典型的以特定品牌作为加分条件。
  - 物业管理服务采购中，设备配置通常作为服务的一部分，但直接指定品牌作为加分项，限制了其他同等质量国产品牌供应商的公平竞争机会。
  - 违反了政府采购中不得以特定品牌、型号作为评审因素的规定，具有明显的指向性和排他性。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；……（八）以其他不合理条件限制或者排斥潜在供应商。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 删除“华为小米等相关国产品牌优先”的表述。
  - 若需鼓励国产品牌，应表述为“设备配置采用国产品牌（需提供相关证明材料）”，且不得指定具体品牌。
  - 若需对设备性能进行评分，应设定具体的技术参数指标（如功能、性能参数等），而非品牌名称。

## 风险点2：评分标准中设置与履约能力关联度不高的企业荣誉及特定荣誉加分

- 问题定性：中风险
- 审查类型：评分因素不相关
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 商务项（F3×A3）-> 1.企业荣誉、8.项目荣誉
- 原文摘录：- “1.企业荣誉：投标人2020年1月1日以来……获得过地市级（含）以上人民政府或行政机关颁发的与物业服务或企业诚信经营有关的荣誉或表彰的，每提供1份得1分，满分3分。”
- “8.项目荣誉：投标人已完成或正在履约的物业管理服务项目……获得国家部委组织的‘节约型公共机构示范单位’或‘节约型机关’创建工作荣誉的得0.5分……获得地市级（含）以上人民政府或行政机关授予‘城区生活垃圾分类工作先进单位’荣誉的得0.5分……获得地市级（含）以上人民政府或行政机关授予‘绿化或园林式单位’荣誉的得0.5分。”
- 风险判断：
  - “企业荣誉”项中“与物业服务或企业诚信经营有关”的表述过于宽泛，缺乏量化标准，且荣誉的获取往往与企业的历史积累、公关能力相关，与本项目具体的履约能力（如人员素质、管理方案、应急响应等）无直接必然联系。
  - “项目荣誉”中要求获得“节约型公共机构示范单位”、“城区生活垃圾分类工作先进单位”等特定荣誉，这些荣誉通常是针对业主单位（采购人）或特定项目的综合评定，而非物业服务企业本身的履约能力证明。将业主单位的荣誉或特定项目的荣誉作为对服务企业的评分项，逻辑上存在偏差，且可能限制未获得此类特定荣誉但具备同等服务能力的中小企业参与竞争。
  - 此类评分因素容易导致“唯荣誉论”，偏离了“择优”的采购目标，可能构成以不合理的条件限制竞争。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。商务条件和采购需求指标有区间规定的，评审因素应当量化到相应区间，并设置各区间对应的不同分值。
  - 《政府采购需求管理办法》（财库〔2021〕22号）第二十一条：采购需求客观、明确，且资金及交易条件允许市场供应的，应当通过综合评分法进行评审。评审因素应当与采购标的的履约能力直接相关。
- 整改建议：
  - 删除“企业荣誉”中关于“诚信经营”等宽泛荣誉的评分项，或将其改为与本项目直接相关的“服务质量管理体系认证”（如ISO9001等）且需与物业管理相关。
  - 删除“项目荣誉”中针对特定荣誉称号（如“节约型公共机构示范单位”等）的评分项。
  - 将评分重点转向与项目履约直接相关的因素，如：类似项目业绩（合同金额、服务内容）、项目团队人员资质、服务方案的可操作性、应急预案的完善程度等。

## 风险点3：评分标准中人员配置要求存在重复计分及过度限制风险

- 问题定性：中风险
- 审查类型：评分因素不相关/限制竞争
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 技术项（F2×A2）-> 17-24项（项目经理至绿化工配置情况）
- 原文摘录：“第17-24评分项提供的人员不得重复，提供重复人员不得分。”
- 风险判断：
  - 招标文件要求项目经理、行政专员、安防主管、工程主管、环境主管、综合主管、工程技术人员、绿化工等8个岗位的人员不得重复。
  - 在物业管理服务中，部分岗位（如工程技术人员、绿化工）可能存在一人多岗或兼任的情况，特别是在中小型物业公司中。强制要求所有关键岗位人员“不得重复”，可能人为提高了投标门槛，排斥了人员配置灵活但具备同等服务能力的中小企业。
  - 虽然招标文件在第五章“人员配备及要求”中规定了具体编制人数，但评分项中强制要求“不得重复”且未说明是否允许在满足编制人数前提下的合理兼职（如绿化工在特定情况下兼顾部分环境工作），可能构成不合理的限制。
  - 需核实招标文件第五章中是否明确禁止兼职。若第五章未禁止，评分项的“不得重复”要求则与采购需求不一致，且可能构成歧视。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；（八）以其他不合理条件限制或者排斥潜在供应商。
  - 《政府采购促进中小企业发展管理办法》（财库〔2020〕46号）：不得以不合理的条件限制中小企业参与政府采购活动。
- 整改建议：
  - 审查第五章“人员配备及要求”中是否明确禁止人员兼职。
  - 若第五章未禁止，建议修改评分标准，允许在满足招标文件规定的人员编制总数和关键岗位（如项目经理、主管）专职要求的前提下，其他辅助岗位人员可合理兼职，或明确“不得重复”的具体范围（如仅限核心管理岗位）。
  - 确保评分标准与采购需求（第五章）保持一致，避免设置过高的非必要门槛。

## 风险点4：商务条款中关于“无犯罪证明”的提交时限及无效投标处理存在法律风险

- 问题定性：中风险
- 审查类型：商务条款失衡/不合理条件
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （二）物业管理团队人员配备情况及要求 -> 2)
- 原文摘录：“投标人须单独提供承诺函（格式自拟），承诺在合同签订后的三个月内提供无犯罪证明，未提供的按无效投标处理。”
- 风险判断：
  - 招标文件要求投标人在“合同签订后的三个月内”提供无犯罪证明，否则“按无效投标处理”。
  - “无效投标”通常指在开标、评标阶段因不符合招标文件实质性要求而被否决投标的情形。将“合同签订后”的履约行为（提供无犯罪证明）作为“无效投标”的处理依据，逻辑上存在矛盾。投标阶段无法预知合同签订后的情况，且“无效投标”的认定应在评标阶段完成。
  - 正确的处理方式应是在合同中约定：若中标人未能在约定时间内提供无犯罪证明，采购人有权解除合同或视为违约，而非追溯认定为“无效投标”。
  - 此外，要求所有拟派人员（包括普通保洁、绿化工等）在中标后三个月内提供无犯罪证明，可能涉及个人隐私保护及实际操作难度，需评估其必要性和合理性。
- 法律/政策依据：
  - 《中华人民共和国政府采购法》第五十条：政府采购合同的双方当事人不得擅自变更、中止或者终止合同。
  - 《中华人民共和国民法典》关于合同解除及违约责任的相关规定。
  - 关于“无效投标”的认定，应严格依据《政府采购货物和服务招标投标管理办法》（财政部令第87号）第六十三条等规定，在评标阶段进行。
- 整改建议：
  - 修改表述，将“按无效投标处理”改为“采购人有权解除合同，并没收履约保证金”或“视为中标人违约，承担相应违约责任”。
  - 明确无犯罪证明的提供范围（如仅限安保、消防等关键岗位人员），避免对所有普通岗位人员提出过高要求。
  - 在合同中明确约定提供无犯罪证明的时限及违约责任，而非在招标文件中将其作为投标无效的条件。

## 风险点5：评分标准中“信息化软件服务能力”要求著作权人为投标人，可能限制竞争

- 问题定性：中风险
- 审查类型：评分因素不相关/限制竞争
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 商务项（F3×A3）-> 6.信息化软件服务能力
- 原文摘录：“投标人拥有与物业服务相关信息化管理软件或系统，并承诺在服务期内该软件或系统会投入本项目使用的，每提供1项得1分，满分3分。投标人须提供承诺函和著作权人为投标人的《计算机软件著作权登记证书》……"
- 风险判断：
  - 要求“著作权人为投标人”的《计算机软件著作权登记证书》作为加分条件，排除了通过合法购买、授权许可方式使用成熟物业管理软件的大型或中型企业。
  - 许多专业的物业管理公司可能使用行业通用的成熟软件（如通过SaaS模式或购买授权），而非自行开发。强制要求“著作权人为投标人”，实际上是将评分优势倾斜给具备软件开发能力或自行开发软件的企业，限制了其他具备同等服务能力但使用成熟软件的企业参与竞争。
  - 该要求与“物业服务能力”无直接必然联系，属于不合理的评分因素。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；（八）以其他不合理条件限制或者排斥潜在供应商。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：不得将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 修改评分标准，将“著作权人为投标人”改为“投标人拥有或合法授权使用与物业服务相关的信息化管理软件或系统”。
  - 允许投标人提供软件著作权证书（著作权人为投标人）或软件购买合同、授权许可协议等证明材料。
  - 重点考察软件的功能是否满足本项目需求，而非软件的权属。

## 风险点6：技术参数中关于“实验服清洗服务”指定品牌及配置要求存在倾向性

- 问题定性：中风险
- 审查类型：技术参数倾向性
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> H.其他服务 -> 2.实验服清洗服务
- 原文摘录：“由中标人提供8台全新滚筒洗衣机（容积大于等于10公斤、洗烘一体，如：美的、海尔、小天鹅同品质品牌）供检验科所使用……"
- 风险判断：
  - 在技术参数中列举“美的、海尔、小天鹅”等具体品牌，并标注“同品质品牌”，虽然使用了“如”字，但在实际操作中容易引导投标人倾向于选择这些品牌，或导致其他品牌（如松下、西门子等同等品质品牌）在评审中被认为“非同品质”而处于劣势。
  - 对于服务类项目，设备配置应侧重于性能参数（如容积、功能、能效等级等），而非具体品牌。
  - 指定品牌可能构成对特定供应商的倾向性，违反公平竞争原则。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 删除“如：美的、海尔、小天鹅同品质品牌”的表述。
  - 明确洗衣机的具体技术参数要求，如：容积≥10公斤、洗烘一体、能效等级≥一级、具备特定消毒功能等。
  - 允许投标人提供满足上述技术参数要求的任何品牌产品。

## 风险点7：商务条款中关于“人员工资”报价要求可能构成隐性门槛

- 问题定性：低风险
- 审查类型：商务条款失衡
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 其他商务要求 -> (1)项目报价要求 -> 1)员工工资方面
- 原文摘录：“投标人须承诺拟派往本项目的服务人员薪资不低于福州市现行最低工资标准……投标人须针对员工薪资及五险一金要求事项单独提供承诺函，并在投标文件报价部分的‘分项报价表’中对‘人员工资’和‘人员五险一金’进行分别报价，否则按无效投标处理。投标人在分项报价表中所报的‘人员工资’和‘人员五险一金’经测算低于福州市规定最低标准的，按无效投标处理。”
- 风险判断：
  - 要求对“人员工资”和“人员五险一金”进行分别报价，并设定“低于福州市规定最低标准”即无效投标，虽然旨在保障劳动者权益，但“分别报价”的要求可能增加投标人的编制难度，且“测算”过程可能存在主观性。
  - 若招标文件未明确“测算”的具体方法和标准，可能导致评标委员会在评审时产生争议。
  - 此外，要求“分别报价”可能暴露投标人的成本结构，不利于商业保密。
  - 但总体来看，该条款旨在落实最低工资标准，符合法律法规要求，风险相对较低，但需确保执行标准明确。
- 法律/政策依据：
  - 《中华人民共和国劳动法》第四十八条：国家实行最低工资保障制度。
  - 《中华人民共和国政府采购法》第二十二条：供应商参加政府采购活动应当具备下列条件：……（二）有依法缴纳税收和社会保障资金的良好记录。
- 整改建议：
  - 明确“测算”的具体方法和标准，如：分项报价表中“人员工资”总额除以拟派人员总数，应不低于福州市最低工资标准；“人员五险一金”总额除以拟派人员总数，应不低于福州市最低缴纳标准。
  - 建议允许投标人在分项报价表中合并报价，但需在投标文件中提供详细的工资及社保缴纳承诺及测算说明，而非强制要求“分别报价”。
  - 确保评审标准客观、量化，避免主观裁量。

## 风险点8：评分标准中“项目业绩”要求“非住宅类”政府采购项目，可能限制中小企业

- 问题定性：低风险
- 审查类型：业绩与证书要求
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 商务项（F3×A3）-> 3.项目业绩、4.履约经验、5.服务满意度
- 原文摘录：“根据投标人提供的自2019年1月1日（含）至本项目投标截止时间止（以验收日期为准）已完成的同类（非住宅类）政府采购项目业绩情况进行评分……”
- 风险判断：
  - 要求业绩必须为“非住宅类”政府采购项目，虽然本项目为研究院物业管理，属于非住宅类，但“政府采购项目”这一限定可能排除了部分具备同等服务能力但主要承接非政府采购项目（如企业、事业单位自行招标项目）的中小企业。
  - 对于中小企业而言，获取政府采购项目业绩的难度相对较大，该要求可能构成隐性门槛。
  - 需核实“非住宅类”是否为本项目服务的必要特征。若本项目服务内容与住宅类物业管理有显著差异，则该要求合理；若差异不大，则可能构成限制。
- 法律/政策依据：
  - 《政府采购促进中小企业发展管理办法》（财库〔2020〕46号）：不得以不合理的条件限制中小企业参与政府采购活动。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：不得将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 建议将“政府采购项目”改为“同类项目”，即“非住宅类物业管理项目”，不限定是否为政府采购项目。
  - 若必须限定为政府采购项目，需提供充分的理由说明其必要性（如本项目涉及特殊的政府采购管理要求）。
  - 确保业绩要求与项目规模、服务内容相匹配，避免设置过高的业绩门槛。

## 综合判断

- 高风险问题：
  - 评分标准中设置特定品牌倾向性条款（风险点1）
- 中风险问题：
  - 评分标准中设置与履约能力关联度不高的企业荣誉及特定荣誉加分（风险点2）
  - 评分标准中人员配置要求存在重复计分及过度限制风险（风险点3）
  - 商务条款中关于“无犯罪证明”的提交时限及无效投标处理存在法律风险（风险点4）
  - 评分标准中“信息化软件服务能力”要求著作权人为投标人，可能限制竞争（风险点5）
  - 技术参数中关于“实验服清洗服务”指定品牌及配置要求存在倾向性（风险点6）
- 需人工复核事项：
  - 评分标准中“项目业绩”要求“非住宅类”政府采购项目，可能限制中小企业（风险点8）：需结合项目实际服务内容及当地中小企业参与政府采购的实际情况进行复核。
