## 风险点1：技术参数中指定特定品牌，具有明显倾向性

- 问题定性：高风险
- 审查类型：技术参数倾向性
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （四）、样品要求 -> 1．供应商需提供以下样品 -> ⑤
- 原文摘录：`⑤品牌:Aokang`
- 风险判断：
  - 招标文件在样品要求中直接指定了具体品牌"Aokang"，未说明该品牌为示例或允许提供同等档次其他品牌。
  - 该要求直接指向特定供应商或特定产品，排斥了其他符合国家标准但品牌不同的潜在供应商。
  - 违反了政府采购应当遵循的公平竞争原则，属于以不合理的条件限制或者排斥潜在供应商。
- 法律/政策依据：
  - 《中华人民共和国政府采购法》第二十二条第二款：采购人可以根据采购项目的特殊要求，规定供应商的特定条件，但不得以不合理的条件对供应商实行差别待遇或者歧视待遇。
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；……
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 删除样品要求中“品牌:Aokang"的指定内容。
  - 修改为“样品需体现施工分层截面结构……品牌不限，但需满足招标文件规定的技术参数及国家标准要求”。
  - 若该品牌为市场主流或具有特殊优势，应在技术参数中明确具体的性能指标，而非直接指定品牌。

## 风险点2：样品评审规则设置不当，存在排斥潜在供应商风险

- 问题定性：中风险
- 审查类型：评审标准/样品要求
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （四）、样品要求 -> 2.样品提供注意事项 -> 2.1
- 原文摘录：`2.1样品需在响应文件递交截止时间前送达开标地点并进行样品签到手续，未提供样品、样品提供不全的样品评审不得分。`
- 风险判断：
  - 本项目采用“最低评标价法”（见第四章 7.1条），评标标准仅依据价格，未设置“样品评审”作为评分项。
  - 在最低评标价法下，设置“样品评审不得分”的表述逻辑混乱，且若将样品作为实质性响应条件（即未提供即无效），则需明确列为“★”号实质性条款。
  - 原文表述“样品评审不得分”暗示存在评分环节，与评标方法（最低评标价法）冲突；若意指“未提供样品视为无效投标”，则表述不严谨，易引发歧义。
  - 此外，要求样品在“开标地点”送达，而本项目采用“远程不见面开标”（见第二章 19其他事项），物理送达要求与远程开标模式存在操作冲突，可能增加供应商负担或导致因送达问题被误判无效。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条：综合评分法，是指投标文件满足招标文件全部实质性要求，且按照评审因素的量化指标评审得分最高的投标人为中标候选人的评标方法。最低评标价法，是指投标文件满足招标文件全部实质性要求且投标报价最低的投标人为中标候选人的评标方法。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十一条：评标委员会应当对符合性审查合格的投标文件进行评审。
- 整改建议：
  - 明确样品在评标中的法律地位：若样品为实质性要求，应明确标注“★”并规定“未提供样品或样品不全的，按无效投标处理”，删除“不得分”等与最低评标价法不符的表述。
  - 若样品仅作为参考，应明确“样品不作为评审依据，仅作为验收参考”。
  - 鉴于本项目为远程开标，建议修改样品提交方式，允许通过快递寄送并在开标前送达指定地点，或明确样品在远程开标环境下的提交与查验流程，避免“送达开标地点”与“远程开标”的冲突。

## 风险点3：质疑受理条件设置不合理，增加供应商维权难度

- 问题定性：中风险
- 审查类型：商务条款/质疑投诉
- 原文位置：第二章 投标人须知前附表 -> 19 其他事项 -> (2) 其他 -> 3、质疑受理的其它要求 -> a
- 原文摘录：`a.质疑人递交质疑函时还应出具质疑人已在福建省政府采购网上公开信息系统上已对本项目进行报名的证明文件（体现报名时间），否则将不被认定为潜在投标人，其质疑将不予受理。【招标文件首次下载之日为质疑人在福建省政府采购网上公开信息系统成功报名之日。】`
- 风险判断：
  - 该条款要求质疑人必须提供“报名证明文件”且需体现“报名时间”，否则不予受理。
  - 虽然系统下载记录可视为报名，但要求供应商额外提供“证明文件”增加了质疑的门槛和程序负担。
  - 根据《政府采购质疑和投诉办法》，质疑人只需证明其是“参与该政府采购活动的供应商”或“潜在供应商”。系统下载记录通常由代理机构后台掌握，要求供应商自行举证“报名时间”可能因系统功能限制导致供应商无法提供，从而变相剥夺其质疑权。
  - 该要求可能构成对供应商质疑权利的不合理限制。
- 法律/政策依据：
  - 《政府采购质疑和投诉办法》（财政部令第94号）第十条：供应商认为采购文件、采购过程、中标或者成交结果使自己的权益受到损害的，可以在知道或者应知其权益受到损害之日起7个工作日内，以书面形式向采购人、采购代理机构提出质疑。
  - 《政府采购质疑和投诉办法》（财政部令第94号）第十一条：提出质疑的供应商应当是参与所质疑项目采购活动的供应商。潜在供应商已依法获取其可质疑的采购文件的，可以对该文件提出质疑。
- 整改建议：
  - 删除“出具……报名证明文件”的强制性要求。
  - 修改为：质疑人应提供其已获取招标文件的证明（如系统下载记录截图、获取时间等），由代理机构在系统中核实，而非要求供应商提供额外的“证明文件”。
  - 确保质疑受理条件符合法律法规，不设置额外障碍。

## 风险点4：履约保证金退还期限表述存在歧义，可能损害供应商权益

- 问题定性：低风险
- 审查类型：商务条款/合同条款
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 7、履约保证金
- 原文摘录：`该履约保证金在整体项目竣工验收合格且无任何违约情形的前提下，15个工作日之内一次性无息退还。`
- 风险判断：
  - 条款规定“15个工作日之内”退还，但未明确起算时间（是验收合格之日？还是验收报告签署之日？还是供应商申请之日？）。
  - 虽然《政府采购法实施条例》规定应在验收合格后及时退还，但“15个工作日”的表述若被理解为“验收合格后15个工作日内”，则符合规定；若理解为“验收合格且无违约后15个工作日内”，则起算点模糊。
  - 更关键的是，未明确“验收合格”的认定标准及流程，若采购人拖延验收，将导致退还期限无限期推迟。
  - 建议明确起算点为“验收合格报告签署之日”或“验收合格确认书出具之日”。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第四十八条：采购文件、采购合同、中标通知书等文件中关于履约保证金的退还期限，应当明确。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第二十八条：招标文件中应当明确履约保证金的退还期限。
- 整改建议：
  - 修改为：“该履约保证金在整体项目竣工验收合格（以验收报告签署之日为准）且无任何违约情形的前提下，采购人应在15个工作日内一次性无息退还。”
  - 明确验收合格的认定依据和起算时间，避免歧义。

## 风险点5：合同文本中关键信息缺失，影响合同效力

- 问题定性：低风险
- 审查类型：合同条款
- 原文位置：第六章 政府采购合同 -> 第一节 政府采购合同协议书 -> 甲方（采购人）信息栏
- 原文摘录：`单位名称（公章或合同章）： {{未填写}}（盖章）` `法定代表人或其委托代理人（签章）：{{未填写}}` `住 所：{{未填写}}` ... `电子邮箱：{{未填写}}`
- 风险判断：
  - 招标文件中提供的合同参考文本，甲方（采购人）的关键信息（名称、住所、联系人、电话、邮箱等）均显示为“{{未填写}}”。
  - 虽然这是合同模板，但在招标文件中直接展示未填写的模板，可能导致中标后合同签署时出现信息遗漏或争议。
  - 建议招标文件中应提供填写完整的合同模板，或明确提示中标后需由双方协商填写，但关键信息（如采购人名称、地址）应在招标文件中明确。
- 法律/政策依据：
  - 《中华人民共和国民法典》第四百七十条：合同的内容由当事人约定，一般包括下列条款：（一）当事人的姓名或者名称和住所；……
- 整改建议：
  - 在招标文件中提供填写完整的合同参考文本，至少应填写采购人名称、地址、联系人等固定信息。
  - 或在招标文件中明确：“本合同模板中‘{{未填写}}’部分，由中标人与采购人在签订合同时根据实际协商填写，但采购人名称、地址等信息以招标文件第一章为准。”

## 风险点6：技术参数中部分指标引用标准版本需核实

- 问题定性：需人工复核
- 审查类型：技术参数
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （二）、技术要求 -> 3.1、3.2、4.1、4.2
- 原文摘录：`须符合中小学合成材料面层运动场地（GB 36246-2018）`
- 风险判断：
  - 招标文件多次引用 GB 36246-2018 标准。需核实该标准是否为现行有效版本，是否有更新版本发布。
  - 若该标准已被新标准替代，引用旧标准可能导致验收依据错误。
  - 需人工复核该标准的最新状态。
- 法律/政策依据：
  - 《中华人民共和国标准化法》：国家标准、行业标准、地方标准、团体标准、企业标准的技术要求不得低于强制性国家标准的相关技术要求。
- 整改建议：
  - 需人工复核 GB 36246-2018 是否为现行有效标准。
  - 若存在更新版本，应修改为最新有效标准版本。
  - 若该标准已废止，应替换为现行有效的国家标准或行业标准。

## 风险点7：强制要求翻译机构为中国翻译协会成员单位，涉嫌设置不合理门槛

- 问题定性：中风险
- 审查类型：资格条件审查
- 原文位置：证据1，第 291 - 293 行
- 原文摘录：前述翻译机构应为中国翻译协会成员单位，翻译的中文译本应由翻译人员签名并加盖翻译机构公章，同时提供翻译人员翻译资格证书。
- 风险判断：
  - 招标文件要求非中文资料必须由‘中国翻译协会成员单位’翻译，将特定行业协会的成员资格作为投标资格条件。
  - 该要求与采购标的的履约能力无直接关联，翻译质量应由翻译人员资质和机构合法性决定，而非是否加入特定协会。
  - 此规定可能排斥了具备合法翻译资质但未加入该协会的其他翻译机构，限制了潜在供应商的选择权，涉嫌以不合理的条件对供应商实行差别待遇或歧视待遇。
  - 根据政府采购法，不得以特定行政区域或者特定行业的业绩、奖项、协会成员资格等作为加分条件或中标条件，也不应作为资格门槛。
  - 建议修改为要求提供具有合法翻译资质的机构出具的译本，并附翻译人员资格证书，不再限定协会成员身份。
- 法律/政策依据：
  - 《中华人民共和国政府采购法》第二十二条：采购人可以根据采购项目的特殊要求，规定供应商的特定条件，但不得以不合理的条件对供应商实行差别待遇或者歧视待遇。
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（八）以其他不合理条件限制或者排斥潜在供应商。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 删除‘前述翻译机构应为中国翻译协会成员单位’的限制性表述。
  - 修改为：‘若原件属于非中文描述，应提供具有合法翻译资质的机构翻译的中文译本，翻译的中文译本应由翻译人员签名并加盖翻译机构公章，同时提供翻译人员翻译资格证书。’

## 风险点8：最低评标价法下设置特定品牌样品要求，涉嫌排斥潜在供应商

- 问题定性：高风险
- 审查类型：合规性审查
- 原文位置：证据1 第 1269 - 1285 行
- 原文摘录：⑤品牌:Aokang ... 5.投标人未按时提交样品，或样品不全，或样品经评委认定为未实质性响应招标文件要求的，为无效投标。
- 风险判断：
  - 招标文件第7.1条明确采用'最低评标价法'，该方法下通常不应设置复杂的评分项，样品一般仅作为符合性审查（合格/不合格）依据。
  - 证据1中样品要求第⑤项明确指定品牌为'Aokang'，这直接指向特定供应商，违反了政府采购法关于不得以特定行政区域或者特定行业的业绩、奖项作为加分条件或者中标、成交条件，以及不得以不合理的条件限制或者排斥潜在供应商的规定。
  - 在最低评标价法中，若将特定品牌样品作为实质性响应要求（无效投标条件），实质上构成了对非该品牌供应商的排斥，导致竞争不充分。
  - 即使样品仅作为符合性审查，指定单一品牌也缺乏合理性，除非能证明该品牌具有不可替代性（通常需论证），否则属于违规设置。
- 法律/政策依据：
  - 《中华人民共和国政府采购法》第二十二条：采购人可以根据采购项目的特殊要求，规定供应商的特定条件，但不得以不合理的条件对供应商实行差别待遇或者歧视待遇。
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；（五）限定或者指定特定的专利、商标、品牌或者供应商。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。
- 整改建议：
  - 删除样品要求中指定的品牌'Aokang'，改为描述样品的技术参数、规格、材质等通用要求，允许供应商提供满足技术要求的任意品牌样品。
  - 重新评估在最低评标价法下设置样品的必要性。若样品仅用于验证技术参数，应明确其作为符合性审查标准（合格/不合格），而非评分项。

## 风险点16：检测报告要求中'CMA且CNAS标志'表述存在歧义，可能增加供应商负担

- 问题定性：中风险

- 风险判断：

  - 本项目采用最低评标价法，但招标文件中设置了样品要求及特定品牌（Aokang）的样品提交要求。在最低评标价法下，样品通常仅作为符合性审查依据，若将样品作为评分项或设置特定品牌样品要求，可能构成以不合理的条件限制或排斥潜在供应商，存在指向特定供应商的风险。此外，样品要求中明确指定品牌'Aokang'，涉嫌违反政府采购公平竞争原则。
  - 招标文件在付款、履约及验收条款上整体遵循了政府采购通用规范，明确了国库集中支付、禁止以内部流程为由迟延付款、履约保证金非现金形式及逾期退还责任等公平条款。争议解决机制提供了协商、调解、仲裁或诉讼的多元路径，未设置不合理的管辖限制。但部分核心商务条款（如具体付款节点、验收期限、违约金比例）被留白至《政府采购合同专用条款》，若专用条款未明确具体标准，可能导致验收‘默示通过’条款被滥用或付款条件模糊，存在需人工复核专用条款具体内容的风险。
  - 招标文件在样品要求中直接指定了特定品牌'Aokang'，存在明显的排他性和倾向性风险；部分技术指标引用标准版本（GB/T 1040.2-2006）可能滞后；检测报告要求中CMA与CNAS标志的表述存在歧义风险；合同条款中关于品牌约定的表述可能违反公平竞争原则。
