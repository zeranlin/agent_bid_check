审查对象：`福州一中高中部12号及13号楼学生宿舍家具采购`

## 风险点1：技术参数引用错误标准及标准号存疑

- 问题定性：高风险
- 审查类型：技术参数倾向性/错误
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> 序号1：组合柜 -> 第8点；序号2：学生宿舍座椅 -> 第8点；序号3：楼梯踏板 -> 第4点
- 原文摘录：- 组合柜第8点：“家具中甲醛、苯、二甲苯和TVOC的释放量以及家具中可迁移有害元素限量需要符合GB 28481-2012《家具中有害物质限量》标准以及GB 48001-2026《汽车车门把手安全技术要求》标准中的要求。”
- 座椅第8点：“家具中甲醛、苯、二甲苯和TVOC的释放量，家具中可迁移有害元素限量，以及领苯二甲酸脂、多环芳烃限量需要符合GB18584-2024《家具中有的害物质限量测量》标准中的要求。”
- 楼梯踏板第4点：“家具中可迁移有害元素限量，以及领苯二甲酸脂、多环芳烃限量需要符合GB18584-2024《家具中有害物质限量》标准中的要求并提供福建省检测机构出具的检测报告。”
- 风险判断：
  - 组合柜技术参数中引用了"GB 48001-2026《汽车车门把手安全技术要求》”，该标准名称与家具采购标的完全无关，属于明显的参数错误，可能导致投标人无法响应或产生歧义。
  - 座椅和楼梯踏板引用的"GB18584-2024"标准号存疑。现行有效的家具有害物质限量标准通常为GB 18584-2001（已被GB 28481-2012等替代或补充），且标准名称中“有的害物质”存在错别字。若该标准号不存在或尚未发布，将导致实质性要求无法执行。
  - 要求提供“福建省检测机构出具的检测报告”作为实质性要求（★），可能构成以特定行政区域限制供应商，排斥了其他具备国家认可资质的检测机构出具的报告。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；（八）以其他不合理条件限制或者排斥潜在供应商。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 立即核实并修正组合柜中引用的“汽车车门把手”标准，删除无关标准。
  - 核实GB 18584-2024标准是否存在及有效性，修正标准号及标准名称中的错别字，建议引用现行有效的国家标准（如GB 28481-2012等）。
  - 删除“福建省检测机构”的地域限制，修改为“具有CMA或CNAS资质的检测机构出具的检测报告”。

## 风险点2：样品制作要求具有排他性及泄露信息风险

- 问题定性：高风险
- 审查类型：技术参数倾向性/限制竞争
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （二）样品要求 -> 第3点、第4点
- 原文摘录：- “3、投标人应采用可去除的安全材料隐去投标人名称及其他可能泄露投标人信息的标志、符号等；同时，在投标文件中，也不得出现样品图样，标识等可能泄露投标人样品的任何信息，否则按无效投标处理。”
- “4、投标人必须将以上样品成品提前组装好，并于开标当日投标截止时间前一次性送达投标现场，按要求摆放好样品。必要时评标现场评委可对所有样品进行测试，若出现复核情况时，评标委员会对样品不作再次评审，样品评审结果不作更改。”
- 风险判断：
  - 要求样品“提前组装好”并“一次性送达”，对于大型组合柜（1910*1750*600mm）而言，运输和现场组装难度极大，且样品仅为“柜门”、“桌面板”等部件，要求“组装好”可能指整体柜体，若指整体柜体则运输成本过高，构成不合理负担。
  - 要求“在投标文件中，也不得出现样品图样，标识等可能泄露投标人样品的任何信息”，但招标文件第五章中包含了大量“参考图”和“样品制作要求附图”，这些附图本身可能具有特定设计特征，若要求投标人完全隐去信息，可能导致投标人无法证明其样品符合图纸要求，或者导致评审专家无法核对样品与图纸的一致性。
  - 样品评审结果“不作更改”且“不作再次评审”，若样品在运输或摆放过程中出现非实质性损坏，可能导致直接废标，缺乏救济机制。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得以不合理的条件对供应商实行差别待遇或者歧视待遇。
  - 《政府采购需求管理办法》（财库〔2021〕22号）：采购需求应当符合法律法规、政府采购政策和国家有关规定，符合政府采购集中采购目录和采购限额标准，符合预算、资产配置标准等。
- 整改建议：
  - 明确样品具体为“部件”还是“整体”，若为部件，应明确无需整体组装，只需提供关键部件样品。
  - 允许投标人在投标文件中提供样品照片或图纸以辅助说明，只要不泄露投标人名称即可，不应完全禁止出现样品图样。
  - 增加样品评审的复核机制，允许对非实质性瑕疵进行澄清或补正。

## 风险点3：商务条款中采购人单方变更权过大且结算方式不明

- 问题定性：中风险
- 审查类型：商务条款失衡
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 其他商务要求 -> 8.3、13.3
- 原文摘录：- 8.3：“签订合同后，采购单位有权根据实际需要对本次采购文件中未明确或未规定的事项作出规定或细微调整，及对各种货物的款式进行细微调整（允许偏离±5%），最终以采购人实际为准，中标人的最终报价不变，如遇采购人特殊需求特殊制作与招标尺寸有较大差异的货物，偏离超过5%，结算单价按调整后面积比例相应换算。”
- 13.3：“①本项目中标后，采购人有权对家具的颜色、款式、尺寸等作出细微调整，中标价不变，中标人不得因此增加任何费用或持有任何异议。②...采购人根据现场的实际情况，有权对所有货物的数量、尺寸和款式进行增加、减少或调整，最终以采购人实际的安装数量为准，变化部分的单价与投标报价对应的单价一致...”
- 风险判断：
  - 条款赋予采购人过大的单方变更权，且规定“中标价不变”、“不得增加任何费用”，未明确变更导致成本增加时的补偿机制。若变更导致成本大幅增加（如材料变更、工艺变更），强制要求中标人承担将导致显失公平，可能引发合同纠纷或履约风险。
  - “允许偏离±5%”的表述模糊，未明确是尺寸、数量还是其他参数的偏离，且“最终以采购人实际为准”缺乏客观标准。
  - 结算单价按“调整后面积比例相应换算”缺乏明确的计算公式，容易在结算时产生争议。
- 法律/政策依据：
  - 《中华人民共和国民法典》第五百四十三条：当事人协商一致，可以变更合同。
  - 《政府采购法》第四十九条：政府采购合同履行中，采购人需追加与合同标的相同的货物、工程或者服务的，在不改变合同其他条款的前提下，可以与供应商协商签订补充合同，但所有补充合同的采购金额不得超过原合同采购金额的百分之十。
- 整改建议：
  - 明确变更的范围和限度，对于超出“细微调整”范围的变更，应约定价格调整机制（如参照市场价或成本加合理利润）。
  - 明确“±5%”的具体含义（如尺寸公差或数量浮动），并约定超出该范围的变更需双方协商签订补充协议。
  - 细化“面积比例换算”的计算公式，确保结算透明。

## 风险点4：履约验收条款中“不邀请投标人验收”及驻厂检查权

- 问题定性：中风险
- 审查类型：商务条款失衡/验收条款
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 序号4、10.4
- 原文摘录：- 序号4：“是否邀请投标人验收：不邀请投标人验收”
- 10.4：“投标人须承诺：若中标，在生产本次采购家具的过程中，采购人可对产品从进料、开胚、酸洗磷化、喷涂、成型等一系列生产程序进行驻厂检查、监督；如果采购人在检查监督过程中发现中标人未按照相关工艺要求进行生产，采购人有权终止合同...”
- 风险判断：
  - “不邀请投标人验收”剥夺了供应商参与验收的权利，不符合《政府采购法》关于验收应当由采购人组织，供应商参与的原则，可能导致验收结果不公，供应商无法对验收结果提出异议。
  - 要求供应商承诺“驻厂检查”且采购人有权“终止合同”，虽然体现了质量管控，但“驻厂检查”的具体标准、频率、费用承担未明确，且“未按照相关工艺要求”定义模糊，可能成为采购人随意终止合同的借口。
- 法律/政策依据：
  - 《政府采购法》第四十一条：采购人或者其委托的采购代理机构应当组织对供应商履约的验收。大型或者复杂的政府采购项目，应当邀请国家认可的质量检测机构参加验收工作。验收方成员应当在验收书上签字，并承担相应的法律责任。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第七十一条：采购人应当及时组织验收。
- 整改建议：
  - 修改为“邀请投标人参与验收”，明确供应商在验收过程中的配合义务及异议权利。
  - 细化驻厂检查的条款，明确检查的频率、标准、费用承担（通常由采购人承担或包含在报价中需明确），并明确“未按照工艺要求”的具体判定标准，避免随意终止合同。

## 风险点5：技术参数中特定工艺及材料要求可能具有倾向性

- 问题定性：中风险
- 审查类型：技术参数倾向性
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> 序号1、2、3
- 原文摘录：- 组合柜：“档条基材采用厚度≥25mm的E1级实木多层板...档条上安装延长线插座（带电源适配器）...插座需要内嵌在档板中...抽屉下柜柜门上安装挂钩锁...锁芯和锁片采用合金压铸成型...”
- 座椅：“座板采用PP新料一次注塑制作...座板连接部件和靠背底部腔体采用榫卯连接...靠背底部腔体中间具有≥8个柱体固定，腔体中间需要内置≥3个螺母...”
- 楼梯踏板：“踏步板表面冲压≥70条凸起条纹...荧光条采用PP新料一次注塑制作...荧光条底部具有≥2个内塞，荧光条需要内嵌在凹槽中，不能采用胶水和螺丝固定。”
- 风险判断：
  - 部分参数描述过于具体，如“榫卯连接”、“内塞卡入”、“不能采用胶水和螺丝固定”等，可能指向特定供应商的专利工艺或特定产品，限制了其他采用不同但等效工艺（如卡扣、焊接等）的供应商参与竞争。
  - “PP新料一次注塑”、“合金压铸成型”等要求虽为常见工艺，但结合具体的尺寸、孔位、内塞数量等细节，可能构成对特定产品的指向。
  - 需核实这些特定工艺是否为行业通用标准，若为特定厂商独有，则构成排他性条款。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（三）采购需求中的技术、服务等要求指向特定供应商、特定产品。
  - 《政府采购需求管理办法》（财库〔2021〕22号）：采购需求应当符合法律法规、政府采购政策和国家有关规定，不得以不合理的条件对供应商实行差别待遇或者歧视待遇。
- 整改建议：
  - 将“榫卯连接”、“内塞卡入”等特定工艺描述修改为功能性描述（如“连接牢固、无外露螺丝、便于拆卸”等），允许采用等效工艺。
  - 删除“不能采用胶水和螺丝固定”等排他性表述，改为“连接方式应牢固、美观、无安全隐患”。
  - 组织专家论证，确认技术参数是否具有排他性。

## 风险点6：履约保证金退还时限及违约责任条款

- 问题定性：低风险
- 审查类型：商务条款失衡
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 序号7、12.3
- 原文摘录：- 序号7：“该履约保证金将在验收合格后且中标人无违约的前提下10个工作日内无息退还（转账方式提交的）或解除担保（保函形式提交的）。”
- 12.3：“中标人未能按时供货，每逾期一天需支付违约金1500元（人民币），超过15天采购人有权解除合同并不予退还履约保证金。”
- 风险判断：
  - 履约保证金退还时限为“验收合格后10个工作日”，符合常规，但需确保验收流程高效，避免无故拖延。
  - 逾期违约金“1500元/天”对于265万元的合同金额而言，比例约为0.056%/天，略高于常规（通常为0.05%/天或0.1%/天），但尚在合理范围内。
  - “超过15天...不予退还履约保证金”条款较为严厉，若逾期原因非中标人主观恶意（如不可抗力），直接没收全部保证金可能显失公平。
- 法律/政策依据：
  - 《政府采购法》第四十六条：采购人与中标、成交供应商应当在中标、成交通知书发出之日起三十日内，按照采购文件确定的事项签订政府采购合同。
  - 《民法典》第五百八十五条：约定的违约金低于造成的损失的，人民法院或者仲裁机构可以根据当事人的请求予以增加；约定的违约金过分高于造成的损失的，人民法院或者仲裁机构可以根据当事人的请求予以适当减少。
- 整改建议：
  - 建议明确“不予退还履约保证金”的适用情形，增加“因不可抗力导致的逾期除外”等免责条款。
  - 确认违约金比例是否合理，避免过高导致合同条款无效。

## 综合判断

- 高风险问题：
  - 技术参数引用错误标准及标准号存疑（风险点1）
  - 样品制作要求具有排他性及泄露信息风险（风险点2）
- 中风险问题：
  - 商务条款中采购人单方变更权过大且结算方式不明（风险点3）
  - 履约验收条款中“不邀请投标人验收”及驻厂检查权（风险点4）
  - 技术参数中特定工艺及材料要求可能具有倾向性（风险点5）
- 需人工复核事项：
  - 需核实GB 18584-2024标准是否存在及有效性。
  - 需核实“榫卯连接”、“内塞卡入”等工艺是否为行业通用标准。
  - 需核实样品“提前组装好”的具体含义及运输可行性。
