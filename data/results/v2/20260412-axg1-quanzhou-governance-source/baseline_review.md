## 风险点1：技术参数中指定特定品牌，具有明显排他性

- 问题定性：高风险
- 审查类型：技术参数倾向性
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （四）、样品要求 -> 1．供应商需提供以下样品 -> ⑤
- 原文摘录：`⑤品牌:Aokang`
- 风险判断：
  - 招标文件在样品要求中直接指定了具体品牌"Aokang"，未说明该品牌为示例或允许提供同等档次其他品牌。
  - 该要求直接指向特定供应商或特定产品，排斥了其他符合国家标准（GB 36246-2018）的合格供应商，严重违反公平竞争原则。
  - 即使该项目为专门面向中小企业采购，也不得在技术参数中指定特定品牌。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 删除样品要求中“品牌:Aokang"的指定内容。
  - 修改为“样品需符合招标文件规定的技术参数及国家标准，品牌不限”或“提供符合要求的样品，品牌由投标人自定”。

## 风险点2：样品评审规则设置不当，存在“未提供即无效”的过度严苛情形

- 问题定性：中风险
- 审查类型：评审标准不合理
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （四）、样品要求 -> 2.2 及 5
- 原文摘录：`2.2所有样品不得体现供应商公司信息，否则按响应无效处理。` `5.投标人未按时提交样品，或样品不全，或样品经评委认定为未实质性响应招标文件要求的，为无效投标。`
- 风险判断：
  - 虽然样品评审是常见做法，但将“样品不全”或“样品经评委认定为未实质性响应”直接作为“无效投标”（即废标）的条件，可能过于严苛。
  - 若样品仅作为评分项（加分项），未提供或样品一般应扣减相应分数，而非直接导致投标无效，除非招标文件明确样品为实质性响应条件且与核心功能直接相关。
  - 本项目采用“最低评标价法”，样品通常用于验证技术参数响应情况。若样品仅用于验证，未提供样品应视为未响应技术参数，但需明确是否属于“实质性要求”。
  - 原文中“样品不全”即无效，若仅缺少非关键样品（如减震垫颜色不符但性能达标），直接废标可能限制竞争。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。
  - 《政府采购法实施条例》第二十条：不得设定与采购项目的具体特点和实际需要不相适应或者与合同履行无关的资格、技术、商务条件。
- 整改建议：
  - 明确样品在评标中的具体作用。若样品仅用于验证技术参数，建议将“未提供样品”或“样品不达标”作为技术部分不得分或扣分处理，而非直接作为无效投标条件，除非该样品是核心产品且无法通过其他方式验证。
  - 若必须作为实质性要求，需在技术参数中明确说明样品是验证核心性能的唯一方式，并细化“样品不全”的具体情形。

## 风险点3：质疑受理条件设置额外门槛，限制供应商救济权利

- 问题定性：中风险
- 审查类型：程序性条款违规
- 原文位置：第二章 投标人须知前附表 -> 一、投标人须知前附表1 -> 19 其他事项 -> (2) 其他 -> 3、质疑受理的其它要求 -> a
- 原文摘录：`a.质疑人递交质疑函时还应出具质疑人已在福建省政府采购网上公开信息系统上已对本项目进行报名的证明文件（体现报名时间），否则将不被认定为潜在投标人，其质疑将不予受理。【招标文件首次下载之日为质疑人在福建省政府采购网上公开信息系统成功报名之日。】`
- 风险判断：
  - 政府采购法规定，潜在投标人或投标人均有权提出质疑。
  - 要求质疑人必须提供“报名证明文件”且以“首次下载之日”为报名日，实际上将“下载招标文件”等同于“报名”，并以此作为质疑的前置条件。
  - 虽然福建省系统通常将下载视为报名，但招标文件中额外强调“出具证明文件”且“否则不予受理”，增加了供应商的举证负担，可能因系统记录问题导致合法质疑被拒，限制了供应商的救济权利。
  - 质疑权是法定权利，不应设置法律之外的额外门槛。
- 法律/政策依据：
  - 《中华人民共和国政府采购法》第五十二条：供应商认为采购文件、采购过程和中标、成交结果使自己的权益受到损害的，可以在知道或者应知其权益受到损害之日起七个工作日内，以书面形式向采购人提出质疑。
  - 《政府采购质疑和投诉办法》（财政部令第94号）第十条：供应商提出质疑应当提交质疑函和必要的证明材料。
- 整改建议：
  - 删除“出具质疑人已在福建省政府采购网上公开信息系统上已对本项目进行报名的证明文件”这一额外要求。
  - 明确只要供应商依法获取了招标文件（以系统记录为准），即具备质疑资格，无需额外提供报名证明。

## 风险点4：招标文件解释权归属条款违规

- 问题定性：中风险
- 审查类型：商务条款失衡
- 原文位置：第二章 投标人须知前附表 -> 一、投标人须知前附表1 -> 19 其他事项 -> (2) 其他 -> 5
- 原文摘录：`5、解释权：采购人或采购代理机构拥有本项目招标文件的最终解释权。`
- 风险判断：
  - 招标文件是合同订立的基础，其解释权属于法律范畴，不能由采购人或代理机构单方面声明拥有“最终解释权”。
  - 该条款排除了监管部门、司法机关对招标文件争议的裁决权，也排除了供应商通过质疑、投诉、诉讼等法定途径维护权益的权利，属于霸王条款。
- 法律/政策依据：
  - 《中华人民共和国政府采购法》及相关实施条例：政府采购活动受法律监督，争议解决应依法进行。
  - 《中华人民共和国民法典》第四百九十六条：提供格式条款一方不合理地免除或者减轻其责任、加重对方责任、限制对方主要权利的，该格式条款无效。
- 整改建议：
  - 删除“采购人或采购代理机构拥有本项目招标文件的最终解释权”条款。
  - 改为“本招标文件的解释及争议解决，依照国家有关法律法规及政府采购相关规定执行”。

## 风险点5：履约保证金退还时限表述存在歧义，可能损害供应商权益

- 问题定性：低风险
- 审查类型：商务条款失衡
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 7、履约保证金
- 原文摘录：`该履约保证金在整体项目竣工验收合格且无任何违约情形的前提下，15个工作日之内一次性无息退还。`
- 风险判断：
  - 条款表述为“15个工作日之内”，未明确起算时间（是验收合格之日？还是验收报告签署之日？）。
  - 虽然“15个工作日”符合一般规定，但“竣工验收合格”是一个过程，若未明确以“验收报告签署日”或“验收合格通知书发出日”为起算点，采购人可能以内部流程为由拖延退还。
  - 建议明确起算节点，以保障供应商权益。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第四十八条：采购文件应当明确履约保证金的退还期限。
  - 《保障中小企业款项支付条例》：机关、事业单位从中小企业采购货物、工程、服务，应当自货物、工程、服务交付之日起30日内支付款项；合同另有约定的，付款期限最长不得超过60日。
- 整改建议：
  - 修改为“自项目竣工验收合格报告签署之日起15个工作日内一次性无息退还”。

## 风险点6：样品要求中“不得体现供应商公司信息”与“样品作为验收样本”存在逻辑冲突

- 问题定性：低风险
- 审查类型：技术参数/验收条款
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （四）、样品要求 -> 2.2 及 3
- 原文摘录：`2.2所有样品不得体现供应商公司信息，否则按响应无效处理。` `3.定标后，成交候选人的样品由采购人封存保留作为验收样本。`
- 风险判断：
  - 要求样品不得体现公司信息是为了保证盲评，但在定标后，样品需作为验收样本。
  - 若样品完全匿名，验收时如何确认该样品确实由中标人提供？若中标人更换了样品，采购人无法通过样品上的标识进行核对。
  - 虽然可以通过封存时的记录来对应，但“不得体现公司信息”可能导致验收环节出现“样品与中标人无关”的争议。
  - 需确认在封存环节是否有其他标识（如编号）来关联中标人。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）：验收应当依据采购合同、采购文件、投标文件等。
- 整改建议：
  - 明确样品在评审阶段需匿名（去除公司信息），但在定标后、封存前，允许中标人在样品上标注项目编号、品目号及中标人名称（或编号），以便验收核对。
  - 或者明确“样品在评审时不得体现公司信息，定标后由中标人补充标识并重新封存”。
