## 风险点1：评分标准中设置特定品牌倾向性条款

- 问题定性：高风险
- 审查类型：评分因素不相关/技术参数倾向性
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 加分项（F4×A4）
- 原文摘录：“设备配置如果有华为小米等相关国产品牌优先。”
- 风险判断：
  - 该条款出现在“加分项”中，直接指定了“华为”、“小米”等具体品牌名称。
  - 在政府采购中，除单一来源采购外，不得指定特定品牌、型号或供应商。
  - 该条款具有明显的排他性和倾向性，限制了其他国产品牌（如海尔、美的、联想等）的公平竞争机会，违反了公平竞争原则。
  - 物业管理服务采购中，设备配置通常作为服务的一部分，不应通过指定品牌进行加分，除非有极特殊的不可替代性论证（此处显然不具备）。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；……
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 删除“华为小米等相关国产品牌优先”的表述。
  - 若需鼓励国货，应表述为“设备配置采用符合国家标准及行业标准的国产品牌”，或仅作为一般性要求，不作为加分项。
  - 若确需对特定设备品牌进行考量，应论证其必要性并采用“或同等档次产品”的表述，且不得直接点名特定品牌。

## 风险点2：评分标准中设置与履约能力无关的特定人员证书要求

- 问题定性：中风险
- 审查类型：评分因素不相关/资格条件设置不当
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 技术项（F2×A2）-> 17.项目经理配置情况
- 原文摘录：“持有政府部门颁发的物业管理师资格证书得1分……具有消防行业技能鉴定机构颁发的四级（含）以上消防设施操作员证的得1分”
- 风险判断：
  - “物业管理师”职业资格已于2016年被国家取消，不再作为准入类或水平评价类职业资格。招标文件仍将其作为评分项，依据已失效的政策，属于设置不合理的评分因素。
  - 虽然消防设施操作员证是必要的，但将其作为项目经理的加分项（而非基本资格）可能合理，但需结合项目实际需求。若项目对消防管理要求极高，可设为资格或技术响应项，但作为“加分”需论证其与项目质量的直接相关性。
  - 主要问题在于“物业管理师”证书已取消，继续要求属于设置无效或过时条件。
- 法律/政策依据：
  - 《关于取消部分职业资格许可和认定事项的决定》（国发〔2016〕13号）：取消物业管理师职业资格许可和认定。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。
- 整改建议：
  - 删除“持有政府部门颁发的物业管理师资格证书”的评分项。
  - 可替换为“具有物业管理相关专业中级及以上职称”或“具有5年以上同类项目管理经验”等与履约能力直接相关的指标。
  - 保留消防设施操作员证要求，但建议核实是否应作为项目经理的必备资格条件而非加分项，以确保项目安全。

## 风险点3：评分标准中设置特定人员学历及证书组合，存在排斥潜在供应商风险

- 问题定性：中风险
- 审查类型：评分因素不相关/限制竞争
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 技术项（F2×A2）-> 17-24项（各岗位配置情况）
- 原文摘录：“具有本科（含）以上学历的得1分……持有政府部门颁发的物业管理师资格证书得1分……具有消防行业技能鉴定机构颁发的四级（含）以上消防设施操作员证的得1分”（多处重复出现）
- 风险判断：
  - 对于“绿化工”、“行政专员”等岗位，强制要求“本科”或“大专”学历并作为加分项，可能超出项目实际服务需求。物业管理服务更看重实操经验和技能，过高的学历要求可能排斥了大量具备丰富经验但学历不高的中小企业或劳务型供应商。
  - 多个岗位（如项目经理、工程主管、综合主管等）同时要求“物业管理师”证书（已取消）和特定消防证书，且要求“不得重复”使用人员，导致对人员资质要求过高，可能只有极少数大型物业公司能满足，构成以不合理条件限制竞争。
  - 要求提供“投标截止时间前六个月任意一个月”的社保证明，虽常见，但结合高学历、多证书要求，进一步提高了门槛。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
  - 《政府采购促进中小企业发展管理办法》（财库〔2020〕46号）：不得设置与采购项目的具体特点和实际需要不相适应的资格、技术、商务条件。
- 整改建议：
  - 重新评估各岗位学历要求的必要性，对于非管理核心岗位（如绿化工、普通保洁、普通保安），建议取消或降低学历要求，改为“具备相关岗位工作经验”或“持有相关上岗证”。
  - 删除已取消的“物业管理师”证书要求。
  - 对于关键岗位（如项目经理、工程主管），保留必要的专业证书要求，但应确保与项目规模、复杂程度相匹配。
  - 明确“人员不得重复”的合理性，若项目确实需要多岗位专职人员，应予以保留，但需确保总人数要求合理。

## 风险点4：商务条款中设置过高的履约保证金及严苛的扣款机制

- 问题定性：中风险
- 审查类型：商务条款失衡
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 4.履约保证金；附件：物业管理服务负面清单和扣款一览表
- 原文摘录：“本采购包履约保证金为合同金额的3.0%”；“非采购人原因，中标人存在欠薪行为……视情节扣除考核基金的10%—100%”；“中标人以任何名义和理由将项目进行转包、分包的。扣除考核基金的100%”；“因中标人原因，发生安全事故的……视情节扣除考核基金的10%—100%”
- 风险判断：
  - 履约保证金3%符合《政府采购法》规定（通常不超过10%），但结合后续严苛的扣款机制，可能加重供应商负担。
  - “负面清单”中大量条款设定“扣除考核基金的10%-100%”或“扣除相应人数工资”，且部分扣款标准（如“每人次扣减物业费人民币10000元”）可能过高，缺乏明确的计算依据，存在显失公平的风险。
  - 部分扣款情形（如“未提供本项目的员工工资发放一览表”）直接导致高额罚款，可能超出合理范围，易引发合同纠纷。
  - 考核基金（每月物业管理费的20%）比例较高，若叠加负面清单扣款，可能导致供应商实际收入大幅波动，影响服务稳定性。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第四十八条：采购文件应当明确履约保证金的数额、缴纳方式、退还条件等。
  - 《民法典》第六条：民事主体从事民事活动，应当遵循公平原则，合理确定各方的权利和义务。
- 整改建议：
  - 重新审视“负面清单”中的扣款标准，确保扣款金额与违约行为造成的实际损失或影响程度相匹配，避免“一刀切”的高额罚款。
  - 明确“考核基金”的扣除上限，避免累计扣款超过合同总额的一定比例。
  - 对于“未提供工资发放一览表”等程序性事项，建议设定整改期限，逾期再行扣款，而非直接高额罚款。
  - 建议引入第三方评估或法律顾问对扣款条款的合理性进行复核。

## 风险点5：技术参数中设置特定设备品牌及配置要求，存在倾向性

- 问题定性：中风险
- 审查类型：技术参数倾向性
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> H.其他服务 -> 2.实验服清洗服务
- 原文摘录：“由中标人提供8台全新滚筒洗衣机（容积大于等于10公斤、洗烘一体，如：美的、海尔、小天鹅同品质品牌）供检验科所使用”
- 风险判断：
  - 虽然使用了“如：美的、海尔、小天鹅同品质品牌”的表述，但列举了三个具体品牌，且限定为“同品质品牌”，在实际评审中容易引导评委倾向于这三个品牌，排斥其他同等品质的国产品牌（如松下、西门子等，虽为外资但在中国生产，或国内其他品牌）。
  - 对于“洗烘一体”、“容积大于等于10公斤”等参数，若未充分论证其必要性，可能构成对特定产品的倾向。
  - 建议改为“容积大于等于10公斤、洗烘一体功能的滚筒洗衣机，品牌不限，但需提供产品合格证及检测报告，确保品质不低于市场主流品牌”。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；……
- 整改建议：
  - 删除“如：美的、海尔、小天鹅”等具体品牌列举。
  - 改为“提供容积大于等于10公斤、洗烘一体功能的滚筒洗衣机，品牌不限，但需满足国家相关质量标准，并提供产品合格证及检测报告”。
  - 若确需指定品牌，应进行单一来源采购论证，否则不得在公开招标中指定。

## 风险点6：评分标准中业绩要求设置过窄，限制中小企业参与

- 问题定性：中风险
- 审查类型：业绩与证书要求/限制竞争
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 商务项（F3×A3）-> 3.项目业绩
- 原文摘录：“根据投标人提供的自2019年1月1日（含）至本项目投标截止时间止（以验收日期为准）已完成的同类（非住宅类）政府采购项目业绩情况进行评分……每提供一份业绩证明材料的得1分，满分3分。”
- 风险判断：
  - 要求“同类（非住宅类）政府采购项目业绩”，且限定为“政府采购项目”，可能将大量具有丰富非政府采购（如企业、事业单位自行采购）物业管理经验的优质供应商排除在外。
  - 虽然项目本身是政府采购，但要求业绩也必须是“政府采购项目”，可能限制了部分中小企业（其业务可能更多集中在非政府采购领域）的参与。
  - 建议扩大业绩范围，接受“同类（非住宅类）物业管理项目业绩”，不限定是否为政府采购项目，只要合同真实有效即可。
- 法律/政策依据：
  - 《政府采购促进中小企业发展管理办法》（财库〔2020〕46号）：不得设置与采购项目的具体特点和实际需要不相适应的资格、技术、商务条件。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：不得将特定行政区域、特定行业的业绩、奖项作为加分条件或者中标、成交条件。
- 整改建议：
  - 修改业绩要求，将“政府采购项目”改为“物业管理项目”，不限定采购方式。
  - 明确“同类（非住宅类）”的定义，避免歧义。
  - 若确需强调政府采购经验，应论证其必要性，并考虑设置较低的权重或作为资格条件而非评分项。

## 风险点7：评分标准中设置“无犯罪证明”作为中标后承诺，存在法律风险

- 问题定性：低风险
- 审查类型：资格条件/商务条款
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （二）物业管理团队人员配备情况及要求 -> 2.两院区人员配备要求 -> 2)
- 原文摘录：“投标人须单独提供承诺函（格式自拟），承诺在合同签订后的三个月内提供无犯罪证明，未提供的按无效投标处理。”
- 风险判断：
  - 要求中标后提供“无犯罪证明”作为合同生效或履约条件，存在法律风险。根据《刑事诉讼法》及相关规定，无犯罪记录证明通常由公安机关出具，且个人有权拒绝提供（除非法律明确规定）。
  - 将“未提供无犯罪证明”直接作为“无效投标”处理，可能违反《政府采购法》关于投标无效情形的规定（通常指投标时不符合资格条件）。
  - 建议改为“投标人承诺拟派人员无犯罪记录，并在中标后配合采购人进行背景调查，若发现拟派人员有犯罪记录，采购人有权要求更换”。
- 法律/政策依据：
  - 《中华人民共和国刑事诉讼法》及相关司法解释：关于无犯罪记录证明的出具和使用规定。
  - 《中华人民共和国政府采购法》第二十二条：供应商参加政府采购活动应当具备的条件。
- 整改建议：
  - 删除“未提供的按无效投标处理”的表述。
  - 改为“投标人承诺拟派人员无犯罪记录，并在中标后配合采购人进行背景调查。若发现拟派人员有犯罪记录，采购人有权要求更换，中标人应在规定时间内更换符合要求的人员”。
  - 明确背景调查的范围和程序，避免侵犯个人隐私。

## 风险点8：评分标准中设置“信息化软件服务能力”要求，存在倾向性

- 问题定性：低风险
- 审查类型：评分因素不相关/技术参数倾向性
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 商务项（F3×A3）-> 6.信息化软件服务能力
- 原文摘录：“投标人拥有与物业服务相关信息化管理软件或系统，并承诺在服务期内该软件或系统会投入本项目使用的，每提供1项得1分，满分3分。投标人须提供承诺函和著作权人为投标人的《计算机软件著作权登记证书》……"
- 风险判断：
  - 要求“著作权人为投标人”的《计算机软件著作权登记证书》，可能排斥了使用成熟第三方软件（如SaaS服务、购买授权软件）的供应商。
  - 物业管理信息化软件并非核心服务，要求投标人必须拥有自主知识产权，可能构成对特定类型供应商（如大型科技公司）的倾向，限制了使用成熟第三方解决方案的中小企业参与。
  - 建议改为“投标人拥有或承诺使用与物业服务相关的信息化管理软件或系统，并承诺在服务期内投入本项目使用”，不限定著作权归属。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 删除“著作权人为投标人”的限制。
  - 改为“投标人拥有或承诺使用与物业服务相关的信息化管理软件或系统，并承诺在服务期内投入本项目使用。投标人须提供承诺函及软件使用授权证明（如软件著作权证书、购买合同、授权书等）”。
  - 明确软件功能要求，而非著作权归属。

## 综合判断

- 高风险问题：
  - 评分标准中设置特定品牌倾向性条款（华为、小米）。
- 中风险问题：
  - 评分标准中设置与履约能力无关的特定人员证书要求（物业管理师）。
  - 评分标准中设置特定人员学历及证书组合，存在排斥潜在供应商风险。
  - 商务条款中设置过高的履约保证金及严苛的扣款机制。
  - 技术参数中设置特定设备品牌及配置要求，存在倾向性。
  - 评分标准中业绩要求设置过窄，限制中小企业参与。
- 需人工复核事项：
  - 未发现
