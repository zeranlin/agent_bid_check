## 风险点1：验收检测及相关部门验收费用表述笼统，存在费用边界不清和潜在转嫁风险

- 问题定性：中风险
- 审查类型：需求合规性 / 验收费用边界审查
- 原文位置：规则条款：内置规则库：需求合规性；需求条款：9验收标准及要求：；6.合同履行
- 原文摘录：规则要求：需求合规性-不得要求中标人承担验收产生的检测费用。

条款内容：9验收标准及要求： 9.1验收标准：中标人所提供的货物必须是制造厂家生产的崭新的未开箱的原包装货物。；所有货物按厂家验收标准（符合国家或行业或地方标准）、招标文件、投标文件及中标人在投标文件中所提供的样品要求等有关内容进行验收。；中标人如有以少充多，以劣充优，以假充真的，采购人有权利不予验收，造成的一切后果由中标人自行承担。
- 风险判断：
  - 规则已明确，不得要求中标人承担验收产生的检测费用。
  - 当前条款将检测、相关部门验收等费用笼统纳入投标总价，但未区分履约自检、试运行成本与验收阶段第三方/法定检测费用。
  - 若其中包含项目验收所需专项检测、第三方检测或法定检测事项，则存在将验收检测费用转嫁给中标人的潜在风险。
  - 该类写法首先表现为费用承担边界不清，其次才可能演化为实际费用转嫁争议。
- 法律/政策依据：
  - 已结合规则库交叉校验与原文条款综合判断。
- 整改建议：
  - 将安装调试、自检、试运行等正常履约成本，与采购验收阶段可能发生的专项检测或法定检测费用区分开。
  - 如项目确需检测，应明确检测类型、承担主体和合规依据，避免仅以“投标总价包括检测、相关部门验收等费用”笼统表述。
  - 对“相关部门验收”涉及的具体部门、流程及费用承担方式作进一步说明。

## 风险点2：技术参数存在错误或异常标准引用，可能导致技术要求失真

- 问题定性：高风险
- 审查类型：技术参数倾向性/错误
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> 序号1：组合柜 -> 第8点；序号2：学生宿舍座椅 -> 第8点；序号3：楼梯踏板 -> 第4点
- 原文摘录：- 组合柜第8点：“家具中甲醛、苯、二甲苯和TVOC的释放量以及家具中可迁移有害元素限量需要符合GB 28481-2012《家具中有害物质限量》标准以及GB 48001-2026《汽车车门把手安全技术要求》标准中的要求。”
- 座椅第8点：“家具中甲醛、苯、二甲苯和TVOC的释放量，家具中可迁移有害元素限量，以及领苯二甲酸脂、多环芳烃限量需要符合GB18584-2024《家具中有的害物质限量测量》标准中的要求。”
- 楼梯踏板第4点：“家具中可迁移有害元素限量，以及领苯二甲酸脂、多环芳烃限量需要符合GB18584-2024《家具中有害物质限量》标准中的要求并提供福建省检测机构出具的检测报告。”
- 风险判断：
  - 组合柜技术参数中引用了"GB 48001-2026《汽车车门把手安全技术要求》”，该标准名称与家具采购标的完全无关，属于明显的参数错误，可能导致投标人无法响应或产生歧义。
  - 座椅和楼梯踏板引用的"GB18584-2024"标准号存疑。现行有效的家具有害物质限量标准通常为GB 18584-2001（已被GB 28481-2012等替代或补充），且标准名称中“有的害物质”存在错别字。若该标准号不存在或尚未发布，将导致实质性要求无法执行。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；（八）以其他不合理条件限制或者排斥潜在供应商。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 立即核实并修正组合柜中引用的“汽车车门把手”标准，删除无关标准。
  - 核实GB 18584-2024标准是否存在及有效性，修正标准号及标准名称中的错别字，建议引用现行有效的国家标准（如GB 28481-2012等）。
  - 删除“福建省检测机构”的地域限制，修改为“具有CMA或CNAS资质的检测机构出具的检测报告”。

## 风险点4：样品要求过细且评审规则失衡，存在样品门槛风险

- 问题定性：高风险
- 审查类型：技术参数倾向性/限制竞争
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （二）样品要求 -> 第3点、第4点；证据2，第792-832行，样品制作要求；证据2，第792-832行，样品要求第3条
- 原文摘录：- “3、投标人应采用可去除的安全材料隐去投标人名称及其他可能泄露投标人信息的标志、符号等；同时，在投标文件中，也不得出现样品图样，标识等可能泄露投标人样品的任何信息，否则按无效投标处理。”
- “4、投标人必须将以上样品成品提前组装好，并于开标当日投标截止时间前一次性送达投标现场，按要求摆放好样品。必要时评标现场评委可对所有样品进行测试，若出现复核情况时，评标委员会对样品不作再次评审，样品评审结果不作更改。”

2、不接受3D打印和雕刻加工的样品。... 1、柜门采用基材≥0.8mm的冷轧钢板。柜门背面纵向具备≥2条，宽度≥30mm的加强条。... 防撞角部分需要内嵌在门板中，螺丝固定。... 拉手：采用PP料注塑制作。拉手主体上下两头圆弧型，中间位置内凹。拉手盖板上下两头圆弧型，拉手盖板需要插入拉手主体中，不能突出。
- 风险判断：
  - 要求样品“提前组装好”并“一次性送达”，对于大型组合柜（1910*1750*600mm）而言，运输和现场组装难度极大，且样品仅为“柜门”、“桌面板”等部件，要求“组装好”可能指整体柜体，若指整体柜体则运输成本过高，构成不合理负担。
  - 要求“在投标文件中，也不得出现样品图样，标识等可能泄露投标人样品的任何信息”，但招标文件第五章中包含了大量“参考图”和“样品制作要求附图”，这些附图本身可能具有特定设计特征，若要求投标人完全隐去信息，可能导致无法证明样品与图纸的对应关系，或者变相要求投标人提供与特定设计完全一致的样品，存在指向特定供应商的风险。
  - 样品评审结果“不作更改”且“不作再次评审”，若样品在运输或摆放过程中出现非实质性损坏，可能导致直接废标，缺乏合理的补救机制。
  - 明确禁止3D打印和雕刻加工，可能排除了部分采用先进制造技术但能完全满足功能需求的供应商，增加了传统工艺样品的制作成本。
  - 对样品细节（如拉手形状、内嵌方式、加强条数量）描述过于具体，若这些细节非核心功能需求，可能变相指向特定供应商的现有产品或工艺，限制竞争。
  - 要求样品必须提前组装好并一次性送达，增加了物流和现场准备成本。
  - 要求隐去所有可能泄露投标人信息的标志，执行难度大，可能导致部分供应商因操作不当被判定为无效投标。
  - “可去除的安全材料”定义模糊，缺乏具体操作指引，易引发争议。
  - 在投标文件中禁止出现样品图样，可能影响评委对样品整体设计的理解，降低评审效率。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得以不合理的条件对供应商实行差别待遇或者歧视待遇。
  - 《政府采购需求管理办法》（财库〔2021〕22号）：采购需求应当符合法律法规、政府采购政策和国家有关规定，符合国家强制性标准，遵循预算、资产和财务等相关管理制度规定，符合采购项目特点和实际需要。
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；（四）以特定行政区域或者特定行业的业绩、奖项作为加分条件或者中标、成交条件；（五）对供应商采取不同的资格审查或者评审标准。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第二十条：采购人或者采购代理机构应当根据采购项目的特点和采购需求编制招标文件。招标文件应当包括采购项目的商务条件、采购需求、投标人的资格条件、投标报价要求、评标方法、评标标准以及拟签订的合同文本等。
- 整改建议：
  - 明确样品具体为“部件”还是“整体”，若为部件，应明确无需整体组装，降低运输和现场操作难度。
  - 允许投标人在投标文件中提供样品照片或图纸以证明响应情况，仅要求实物样品隐去品牌标识，而非禁止在文件中出现任何图样。
  - 增加样品评审的容错机制，对于非实质性瑕疵允许澄清或补正。
  - 建议将样品制作要求聚焦于核心功能和质量指标，避免对非必要的工艺细节（如拉手具体形状、内嵌方式）进行过度限定。
  - 若3D打印或雕刻加工能满足功能和质量要求，建议取消禁止性规定，或明确说明禁止的原因（如耐久性不足）。
  - 明确样品制作的具体尺寸和公差范围，避免模糊描述导致供应商理解偏差。
  - 明确“可去除的安全材料”的具体类型和使用方法，提供操作指引。
  - 允许在投标文件中提供样品的非标识性图样（如结构图、尺寸图），以便评委全面了解样品设计。
  - 加强对投标人的培训，确保其正确理解并执行盲样要求。

## 风险点5：商务条款中采购人单方变更权过大且结算方式不明

- 问题定性：中风险
- 审查类型：商务条款失衡
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 其他商务要求 -> 8.3、13.3
- 原文摘录：- 8.3：“签订合同后，采购单位有权根据实际需要对本次采购文件中未明确或未规定的事项作出规定或细微调整，及对各种货物的款式进行细微调整（允许偏离±5%），最终以采购人实际为准，中标人的最终报价不变，如遇采购人特殊需求特殊制作与招标尺寸有较大差异的货物，偏离超过5%，结算单价按调整后面积比例相应换算。”
- 13.3：“①本项目中标后，采购人有权对家具的颜色、款式、尺寸等作出细微调整，中标价不变，中标人不得因此增加任何费用或持有任何异议。②...采购人根据现场的实际情况，有权对所有货物的数量、尺寸和款式进行增加、减少或调整，最终以采购人实际的安装数量为准，变化部分的单价与投标报价对应的单价一致...”
- 风险判断：
  - 条款赋予采购人过大的单方变更权，允许对“未明确或未规定的事项”作出规定，且“最终以采购人实际为准”，这可能导致合同实质性内容在签订后被随意变更，违背了政府采购合同的严肃性。
  - “细微调整（允许偏离±5%）”的定义模糊，且要求“中标价不变”，若调整导致成本增加（如材料变更、工艺复杂化），供应商将承担亏损风险，显失公平。
  - 结算单价按“调整后面积比例相应换算”缺乏明确的计算公式和依据，容易引发合同纠纷。
- 法律/政策依据：
  - 《中华人民共和国民法典》第五百四十三条：当事人协商一致，可以变更合同。
  - 《中华人民共和国政府采购法》第四十九条：政府采购合同履行中，采购人需追加与合同标的相同的货物、工程或者服务的，在不改变合同其他条款的前提下，可以与供应商协商签订补充合同，但所有补充合同的采购金额不得超过原合同采购金额的百分之十。
- 整改建议：
  - 删除“对未明确或未规定的事项作出规定”的表述，明确变更需双方协商一致。
  - 明确“细微调整”的范围和标准，若调整导致成本变化，应约定合理的单价调整机制，而非强制“中标价不变”。
  - 细化面积比例换算的具体公式，确保结算依据清晰、可操作。

- 审查类型：商务条款失衡
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 其他商务要求 -> 10.4
- 原文摘录：- “10.4 投标人须承诺：若中标，在生产本次采购家具的过程中，采购人可对产品从进料、开胚、酸洗磷化、喷涂、成型等一系列生产程序进行驻厂检查、监督；如果采购人在检查监督过程中发现中标人未按照相关工艺要求进行生产，采购人有权终止合同，并按照国家相关法律法规的规定提请有关政府监管部门对其进行处理。”
- 风险判断：
  - 要求采购人进行“驻厂检查、监督”且涵盖“进料、开胚、酸洗磷化、喷涂、成型”等全流程，对于家具制造而言，这属于过度干预供应商的生产经营自主权，且增加了供应商的接待和管理成本。
  - 仅因“未按照相关工艺要求”即可“终止合同”，未区分一般瑕疵与根本违约，处罚措施过重，缺乏比例原则。
- 法律/政策依据：
  - 《中华人民共和国政府采购法》第四十三条：政府采购合同适用合同法。采购人和供应商之间的权利和义务，应当按照平等、自愿的原则以合同方式约定。
  - 《政府采购需求管理办法》（财库〔2021〕22号）：履约验收条款应当明确验收标准、验收程序、验收方法等，不得设置不合理的验收条件。
- 整改建议：
  - 将“驻厂检查”修改为“不定期抽查”或“关键节点检查”，明确检查的频率和范围，避免过度干扰生产。
  - 细化“未按照相关工艺要求”的具体情形，区分一般违约和根本违约，仅对严重影响产品质量或无法整改的情形约定终止合同的权利。

## 风险点7：验收标准引用“厂家验收标准”及“样品”，存在模糊表述和单方裁量风险

- 问题定性：中风险
- 审查类型：标准客观性审查
- 原文位置：证据1，第 9.1 条
- 原文摘录：9.1验收标准：...所有货物按厂家验收标准（符合国家或行业或地方标准）、招标文件、投标文件及中标人在投标文件中所提供的样品要求等有关内容进行验收。
- 风险判断：
  - ‘厂家验收标准’通常低于或不同于国家/行业标准，将其作为验收依据可能导致验收标准降低，且不同厂家标准不一，缺乏统一客观性。
  - ‘样品要求’作为验收标准，若样品本身未封存或标准描述不清，易引发争议，且样品可能无法完全代表批量生产产品的质量。
  - 条款表述‘等有关内容’过于宽泛，赋予采购人过大的自由裁量权，可能导致验收结果的主观性。
  - 未明确当厂家标准与国家标准冲突时，以哪个标准为准，存在标准适用混乱的风险。
- 法律/政策依据：
  - 《政府采购法实施条例》第三十一条：采购文件应当明确采购标的的技术、服务、安全标准。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第 87 号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。
- 整改建议：
  - 删除‘厂家验收标准’作为独立验收依据的表述，明确验收标准应严格遵循国家强制性标准、推荐性标准或行业标准。
  - 若必须参考样品，应明确样品的封存程序及样品作为验收依据的具体范围（如外观、材质等），并规定样品与批量产品的一致性要求。
  - 细化验收标准，列明具体的技术参数、质量指标，减少模糊表述。

- 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第二十条：采购人或者采购代理机构应当根据采购项目的特点和采购需求编制招标文件。招标文件应当包括采购项目的商务条件、采购需求、投标人的资格条件、投标报价要求、评标方法、评标标准以及拟签订的合同文本等。
- 《中华人民共和国民法典》第五百四十三条：当事人协商一致，可以变更合同。
- 《中华人民共和国政府采购法》第四十九条：政府采购合同履行中，采购人需追加与合同标的相同的货物、工程或者服务的，在不改变合同其他条款的前提下，可以与供应商协商签订补充合同，但所有补充合同的采购金额不得超过原合同采购金额的百分之十。
- 《中华人民共和国政府采购法》第四十三条：政府采购合同适用合同法。采购人和供应商之间的权利和义务，应当按照平等、自愿的原则以合同方式约定。
- 《政府采购需求管理办法》（财库〔2021〕22号）：履约验收条款应当明确验收标准、验收程序、验收方法等，不得设置不合理的验收条件。
- 《政府采购法实施条例》第三十一条：采购文件应当明确采购标的的技术、服务、安全标准。
- 《政府采购货物和服务招标投标管理办法》（财政部令第 87 号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。
