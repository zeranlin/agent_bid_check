## 风险点1：评分标准中设置特定品牌倾向性条款

- 问题定性：高风险
- 审查类型：评分因素不相关/技术参数倾向性
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 加分项（F4×A4）
- 原文摘录：“设备配置如果有华为小米等相关国产品牌优先。”
- 风险判断：
  - 该条款出现在“加分项”中，直接指定了“华为”、“小米”等特定品牌名称。
  - 在政府采购中，除单一来源采购外，不得指定特定品牌、型号或供应商。
  - 该条款具有明显的排他性和倾向性，限制了其他国产品牌（如海尔、美的、联想等）的公平竞争机会，违反了公平竞争原则。
  - 物业管理服务采购中，设备配置通常作为服务的一部分，不应通过指定品牌来作为加分项，除非有极特殊的理由且经过论证，但直接列出品牌名称显然违规。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；……
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 删除“华为小米等相关国产品牌优先”的表述。
  - 如确需鼓励国产品牌，应表述为“设备配置采用符合国家标准及行业标准的国产品牌”，或仅作为一般性要求，不得作为加分项指向特定品牌。
  - 若该加分项无实际意义，建议直接删除该加分项。

## 风险点2：评分标准中设置与履约能力无关的特定证书要求

- 问题定性：中风险
- 审查类型：评分因素不相关
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 技术项（F2×A2）-> 17.项目经理配置情况
- 原文摘录：“持有政府部门颁发的物业管理师资格证书得1分”
- 风险判断：
  - “物业管理师”职业资格已于2016年被国家取消（人社部发〔2016〕11号），不再作为准入类或水平评价类职业资格进行统一考试和发证。
  - 虽然部分行业协会或培训机构可能颁发相关证书，但招标文件要求“政府部门颁发”的物业管理师证书，目前实际上已不存在此类政府颁发的证书。
  - 该评分项可能导致所有投标人均无法得分，或者迫使投标人提供非政府颁发的证书（若评委放宽标准），造成评审标准混乱或事实上的无效评分。
  - 该要求与项目实际履约能力无直接必然联系，且依据已失效的政策，属于设置不合理条件。
- 法律/政策依据：
  - 《关于取消部分职业资格许可和认定事项的决定》（国发〔2016〕11号）：取消物业管理师职业资格许可和认定。
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（二）设定的资格、技术、商务条件与采购项目的具体特点和实际需要不相适应或者与合同履行无关；
- 整改建议：
  - 删除“持有政府部门颁发的物业管理师资格证书”的评分项。
  - 可改为考察项目经理的“物业管理相关从业经验”、“持有有效的消防设施操作员证”、“持有有效的特种设备安全管理和作业人员证”等与项目实际管理需求直接相关且现行有效的证书或经验。

## 风险点3：评分标准中设置特定行业荣誉作为加分项，存在排他性风险

- 问题定性：中风险
- 审查类型：评分因素不相关/限制竞争
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 商务项（F3×A3）-> 8.项目荣誉
- 原文摘录：“投标人已完成或正在履约的物业管理服务项目在投标人服务期间获得国家部委组织的‘节约型公共机构示范单位’或‘节约型机关’创建工作荣誉的得0.5分……获得地市级（含）以上人民政府或行政机关授予‘城区生活垃圾分类工作先进单位’荣誉的得0.5分……获得地市级（含）以上人民政府或行政机关授予‘绿化或园林式单位’荣誉的得0.5分。”
- 风险判断：
  - 这些荣誉通常是授予“单位”（即采购人或使用单位）的，而非授予“物业服务企业”的。虽然招标文件要求提供“项目物业服务合同”作为证明，暗示是项目获得的荣誉，但此类荣誉的授予对象往往具有特定性（如特定的机关单位）。
  - 将特定类型的荣誉（如“节约型公共机构示范单位”）作为加分项，可能只有服务过特定类型单位（如政府机关、事业单位）的供应商才能得分，而服务过企业、学校等其他类型单位的优质供应商可能无法获得此类荣誉，从而构成对潜在供应商的差别待遇。
  - 荣誉的获取受多种因素影响，不完全取决于物业服务水平，将其作为评分因素与“合同履行能力”的相关性较弱。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（二）设定的资格、技术、商务条件与采购项目的具体特点和实际需要不相适应或者与合同履行无关；
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。
- 整改建议：
  - 建议删除此类特定荣誉的加分项，或将其修改为更通用的荣誉，如“获得地市级以上人民政府或行政机关颁发的与物业管理服务相关的荣誉（如优秀物业服务企业、物业管理示范项目等）”。
  - 确保评分因素与物业服务水平直接相关，而非与采购人单位的性质或特定荣誉挂钩。

## 风险点4：评分标准中要求提供“非投标人自证”的满意度证明，操作难度大且易引发争议

- 问题定性：中风险
- 审查类型：评分因素不相关/评分标准不明确
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 商务项（F3×A3）-> 5.服务满意度
- 原文摘录：“④非投标人自证的满意度证明材料复印件。所有材料缺一不可，否则不得分。”
- 风险判断：
  - 要求提供“非投标人自证”的满意度证明，通常指由采购人出具的证明。但在投标阶段，投标人无法控制前手采购人是否愿意出具此类证明，或者前手采购人可能因内部流程、人员变动等原因无法及时出具。
  - 该要求可能导致部分有良好履约记录但因客观原因无法提供证明的投标人失分，造成不公平。
  - “非投标人自证”的表述不够明确，容易在评审中产生歧义（例如：第三方机构出具的报告是否算？）。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。评审因素应当在招标文件中规定。
- 整改建议：
  - 明确“非投标人自证”的具体形式，例如：“由原采购人出具的书面满意度评价证明（需加盖公章）”或“第三方专业机构出具的满意度调查报告”。
  - 考虑到实际操作难度，建议允许提供“合同验收合格证明”或“无投诉证明”等作为替代，或者降低该评分项的权重。
  - 明确若因原采购人原因无法提供证明的，投标人可提供其他佐证材料（如合同、验收报告、无投诉声明等）供评委酌情认定。

## 风险点5：技术参数中设置特定人员性别、身高要求，涉嫌就业歧视

- 问题定性：高风险
- 审查类型：资格条件/技术参数倾向性
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （二）物业管理团队人员配备情况及要求 -> 1.两院区人员配备情况 -> 通湖路院区人员配置及要求 -> 行政专员
- 原文摘录：“女性，身体健康，仪表端庄，热情大方，口齿伶俐，普通话标准，身高1.60米以上。”
- 风险判断：
  - 招标文件明确要求行政专员必须为“女性”，且身高“1.60米以上”。
  - 根据《就业促进法》及相关规定，除国家规定的不适合妇女的工种或者岗位外，不得以性别为由拒绝录用妇女或者提高对妇女的录用标准。
  - 物业管理中的行政专员岗位，通常不涉及国家规定的禁止女性从事的工种，设置性别限制属于就业歧视，限制了男性供应商的公平竞争机会。
  - 身高要求（1.60米以上）与行政专员的履职能力无直接必然联系，属于不合理的身体条件限制。
- 法律/政策依据：
  - 《中华人民共和国就业促进法》第二十七条：国家保障妇女享有与男子平等的劳动权利。用人单位招用人员，除国家规定的不适合妇女的工种或者岗位外，不得以性别为由拒绝录用妇女或者提高对妇女的录用标准。
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；（四）以其他不合理条件限制或者排斥潜在供应商。
- 整改建议：
  - 删除“女性”的性别限制，改为“男女不限”。
  - 删除“身高1.60米以上”的身高限制，或改为“形象气质良好，符合岗位工作要求”等更通用的描述。
  - 保留“身体健康，仪表端庄，热情大方，口齿伶俐，普通话标准”等与岗位履职能力直接相关的要求。

## 风险点6：商务条款中设置过高的违约金和扣款标准，可能导致合同权利义务失衡

- 问题定性：中风险
- 审查类型：商务条款失衡
- 原文位置：第五章 招标内容及要求 -> （五）质量考核办法 -> 附件：物业管理服务负面清单和扣款一览表
- 原文摘录：“未提供本项目的员工工资发放一览表的按每人次扣减物业费人民币10000元”、“因中标人管理不善导致花草树木损坏或死亡，绿化的更新或补植由中标人负责，费用由中标人承担……因中标人原因未按采购人提供盆栽租摆服务的扣款100元/盆”、“未按采购人要求进行外墙清洗的视情节扣款1000元-30000元”
- 风险判断：
  - 部分扣款标准（如每人次扣减10000元）相对于物业服务费总额（900万元/3年，即300万元/年）而言，比例较高，且未明确“每人次”的具体定义（是每人每月还是每人每次？）。
  - 外墙清洗扣款上限30000元，若清洗面积较大，该金额可能不足以覆盖实际损失，但若清洗面积较小，该金额则过高，缺乏合理性依据。
  - 虽然招标文件允许采购人根据考核结果扣款，但扣款标准应合理、适度，与违约行为的严重程度相匹配。过高的扣款标准可能导致中标人因轻微过失而面临巨额损失，造成合同权利义务失衡，也可能导致供应商在报价时预留过高的风险金，增加采购成本。
- 法律/政策依据：
  - 《中华人民共和国民法典》第五百八十五条：当事人可以约定一方违约时应当根据违约情况向对方支付一定数额的违约金，也可以约定因违约产生的损失赔偿额的计算方法。约定的违约金低于造成的损失的，人民法院或者仲裁机构可以根据当事人的请求予以增加；约定的违约金过分高于造成的损失的，人民法院或者仲裁机构可以根据当事人的请求予以适当减少。
  - 《政府采购法》第四十三条：政府采购合同适用合同法。采购人和供应商之间的权利和义务，应当按照平等、自愿的原则以合同方式约定。
- 整改建议：
  - 重新评估扣款标准的合理性，确保扣款金额与违约行为造成的实际损失或影响程度相匹配。
  - 明确“每人次”的具体含义，避免歧义。
  - 对于外墙清洗等大额扣款项，建议设定更详细的分级扣款标准，或明确扣款上限不超过该单项服务费用的比例。
  - 建议增加“扣款总额不超过当月物业费的一定比例（如20%）”的兜底条款，防止因累计扣款导致中标人无法继续履约。

## 风险点7：评分标准中要求提供“无犯罪证明”作为中标后承诺，存在法律风险

- 问题定性：中风险
- 审查类型：资格条件/商务条款
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （二）物业管理团队人员配备情况及要求 -> 2.两院区人员配备要求 -> 2)
- 原文摘录：“投标人须单独提供承诺函（格式自拟），承诺在合同签订后的三个月内提供无犯罪证明，未提供的按无效投标处理。”
- 风险判断：
  - 要求投标人在中标后三个月内提供“无犯罪证明”，并将“未提供”作为“无效投标”处理，存在逻辑矛盾。无效投标应在评标阶段认定，而非中标后。
  - 根据《公安机关办理犯罪记录查询工作规定》，个人犯罪记录查询通常由本人申请，且需本人到场或提供授权。要求中标人（企业）在中标后三个月内提供所有拟派人员的无犯罪证明，操作难度大，且可能涉及隐私保护问题。
  - 将“未提供无犯罪证明”直接作为“无效投标”处理，属于在评标阶段无法核实的事项，不符合评标程序。
- 法律/政策依据：
  - 《中华人民共和国政府采购法》第二十二条：供应商参加政府采购活动应当具备下列条件：……（五）参加政府采购活动前三年内，在经营活动中没有重大违法记录。
  - 《公安机关办理犯罪记录查询工作规定》：犯罪记录查询应当由本人申请，或者由有关单位依法申请。
- 整改建议：
  - 将“未提供的按无效投标处理”修改为“未提供的，采购人有权要求中标人限期整改；逾期未整改的，采购人有权取消其中标资格，并没收履约保证金”。
  - 明确“无犯罪证明”的提供方式，例如：由拟派人员本人提供，或由中标人提供相关查询结果。
  - 建议将“无犯罪证明”作为合同履约过程中的管理要求，而非中标后的强制性条件，或者在评标阶段要求投标人承诺拟派人员无犯罪记录，并在中标后提供相关证明。

## 风险点8：评分标准中要求提供“员工工资发放一览表”作为中标后承诺，存在法律风险

- 问题定性：中风险
- 审查类型：商务条款
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （二）物业管理团队人员配备情况及要求 -> 2.两院区人员配备要求 -> 6)
- 原文摘录：“中标人签订合同之日起七个自然日内提供本项目的员工工资发放一览表，投标人须单独提供承诺函，未提供的按无效投标处理。”
- 风险判断：
  - 与风险点7类似，将“未提供员工工资发放一览表”作为“无效投标”处理，存在逻辑矛盾。无效投标应在评标阶段认定。
  - 员工工资发放一览表属于企业内部财务信息，要求中标人在合同签订后7日内提供，可能涉及商业秘密保护问题。
  - 虽然招标文件要求投标人承诺薪资不低于最低工资标准，但要求提供详细的工资发放一览表作为中标后的强制性条件，可能过于严苛。
- 法律/政策依据：
  - 《中华人民共和国政府采购法》第四十三条：政府采购合同适用合同法。采购人和供应商之间的权利和义务，应当按照平等、自愿的原则以合同方式约定。

## 风险点11：工程技术人员证书要求叠加，设置不合理门槛

- 问题定性：高风险
- 审查类型：人员资格与证书要求审查
- 原文位置：证据1，第 716 - 721 行
- 原文摘录：至少有1人具有市场监督管理部门颁发的特种设备安全管理和作业人员证（项目代码：A)或特种设备安全管理人员证(项目代码:A1或A4))的得1分；该人员同时持有有效的应急管理部门或安全监督管理部门颁发的高压电工作业证的得1分...该人员同时还持有有效的应急管理部门或安全监督管理部门颁发的低压电工作业证的得1分
- 风险判断：
  - 要求同一人员同时持有特种设备证、高压电工证、低压电工证三个不同类别的证书才能得满分，属于将多个证书要求叠加在单个人身上。
  - 虽然单个证书可能与项目相关，但要求一人多证且作为评分项，实质上提高了人员获取难度，可能排斥拥有多名持证人员但无一人同时持有多证的中小企业或潜在供应商。
  - 该要求未充分说明为何必须由同一人持有所有证书，而非由团队内不同人员分别持有，存在以不合理条件限制竞争的风险。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；（八）以其他不合理条件限制或者排斥潜在供应商。

- 整改建议：
  - 建议修改评分标准，允许投标人提供多名工程技术人员，只要团队内有人持有相应证书即可得分，无需同一人同时持有所有证书。
  - 或者将证书要求拆分为独立评分项，分别对持有特种设备证、高压证、低压证的人员进行评分，不强制要求同一人持有。

- 整改建议：

## 风险点17：无预付款且采用‘先服务后付款’模式，供应商资金压力较大

- 问题定性：中风险
- 审查类型：商务条款公平性审查
- 原文位置：证据3，第 2625 - 2626 行
- 原文摘录：采用“先服务后付款”的方式，根据采购人对中标人的月考核结果按月支付物业费...采购人收到发票后将于15日内将上一个月的费用汇入中标人指定的账户
- 风险判断：
  - 条款明确约定无预付款，且采用按月后付方式，供应商需先行垫付全部服务成本。
  - 对于物业服务类项目，若服务周期长、金额大，无预付款可能增加中小企业资金周转压力，影响履约能力。
  - 虽然符合部分服务类项目惯例，但需结合项目具体金额评估是否构成对供应商的不合理负担。
- 法律/政策依据：
  - 《政府采购促进中小企业发展管理办法》（财库〔2020〕46号）关于预付款的相关规定（鼓励预付款）
  - 《优化营商环境条例》关于不得设置不合理交易条件的规定
- 整改建议：
  - 建议根据项目规模及供应商承受能力，考虑设置一定比例的预付款（如10%-30%），或缩短付款周期。
  - 若确需后付，应明确考核标准及扣款上限，避免无限期拖延付款。

## 风险点18：履约保证金退还条件中‘完全履约’表述模糊，存在单方裁量风险

- 问题定性：中风险
- 审查类型：履约与验收条款审查
- 原文位置：证据4，第 2613 - 2616 行
- 原文摘录：在合同期满中标人完全履约且将设备、档案完整移交给采购人后5个工作日内予以无息退还
- 风险判断：
  - ‘完全履约’属于主观性较强的模糊表述，缺乏客观量化标准。
  - 采购人可能利用该模糊条款，以细微瑕疵为由拒绝或延迟退还保证金，导致供应商权益受损。
  - 未明确‘完全履约’的具体认定依据（如是否以最终验收报告为准），易引发争议。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第四十八条关于履约保证金退还的规定
  - 《优化营商环境条例》关于规范保证金退还时限和条件的要求
- 整改建议：
  - 建议将‘完全履约’修改为‘经采购人验收合格’或‘合同期满且无未结违约事项’等客观标准。
  - 明确退还保证金的具体触发条件和所需材料清单。

## 风险点19：质疑程序中对证据来源合法性要求过于严苛，可能构成不合理门槛

- 问题定性：中风险
- 审查类型：程序性条款审查
- 原文位置：证据1，第 393 - 412 行
- 原文摘录：若质疑的具体事项按照有关法律、法规和规章规定处于保密阶段，则应提供信息或证明材料为合法或公开渠道获得的有效证据（若证据无法有效表明信息或证明材料为合法或公开渠道获得，则前述信息或证明材料视为无效）。
- 风险判断：
  - 要求质疑人证明证据来源的‘合法性’或‘公开渠道’，对于处于保密阶段的采购信息，供应商往往难以自证。
  - 该条款实质上提高了质疑门槛，可能导致供应商因无法提供‘来源证明’而丧失质疑权利。
  - ‘视为无效’的表述过于绝对，未给予质疑人补充说明或解释的机会。
- 法律/政策依据：
  - 《政府采购质疑和投诉办法》（财政部令第94号）关于质疑受理条件的规定
  - 《中华人民共和国政府采购法》第五十二条关于供应商质疑权利的保护
- 整改建议：
  - 建议删除或修改关于‘证据来源合法性证明’的强制性要求，改为由采购人核实证据真实性。
  - 明确对于保密信息的质疑，采购人应依法履行保密义务并受理，而非直接以证据来源不明为由驳回。

## 风险点20：合同争议解决及违约责任条款引用笼统，缺乏具体约定

- 风险判断：
  - 条款仅笼统引用《民法典》，未明确具体的争议解决方式（如协商、调解、仲裁或诉讼）及管辖地。
  - 缺乏具体的违约责任计算标准（如违约金比例、赔偿范围），可能导致争议发生时双方扯皮。

- 整改建议：
  - 建议在合同中明确争议解决的具体方式（如向采购人所在地人民法院起诉）及管辖法院。
  - 补充具体的违约责任条款，明确逾期付款、逾期履约等情形的违约金计算标准。

## 风险点21：合同续签条件中‘无不良记录’及‘考核合格’标准未量化

- 问题定性：需人工复核
- 审查类型：履约期限与续签审查
- 原文位置：证据3，第 2625 - 2626 行
- 原文摘录：中标人在第1年合同履行期间，工作认真负责、无不良记录，经采购人考核合格后双方续签第2年合同
- 风险判断：
  - ‘工作认真负责’、‘无不良记录’、‘考核合格’等表述主观性强，缺乏量化指标。
  - 采购人可能利用模糊标准随意决定是否续签，导致供应商预期利益落空。
  - 需结合招标文件其他部分（如考核评分表）确认是否有具体标准，若全文无量化标准则风险较高。
- 法律/政策依据：
  - 《中华人民共和国政府采购法》关于公平竞争原则的规定
  - 《优化营商环境条例》关于不得设置不合理交易条件的要求
- 整改建议：
  - 建议明确‘无不良记录’的具体定义（如未发生安全事故、未收到有效投诉等）。
  - 细化‘考核合格’的具体评分标准或分数线，确保续签条件公开透明、可量化。

## 风险点22：绿化存活率指标表述需核实一致性

- 问题定性：低风险
- 审查类型：技术参数一致性审查
- 原文位置：证据2，第1328-1866行
- 原文摘录：1.负责提供园林式绿化服务...绿化成活率98%以上。... 1.除不可抗力和人为因素外，要求室内外绿植存活率达到98%以上。
- 风险判断：
  - 原文在‘服务内容’中表述为‘绿化成活率98%以上’，在‘服务要求’中表述为‘存活率达到98%以上’。
  - 虽然数值均为98%，但‘成活率’与‘存活率’在专业术语上可能存在细微差别，且两处表述位置不同，需人工复核是否构成技术指标前后矛盾或歧义。
  - 若为同一指标，建议统一术语表述，避免评审争议。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（三）采购需求中的技术、服务等要求指向特定供应商、特定产品。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。
- 整改建议：
  - 建议统一全文中关于绿化植物存活/成活率的术语表述，确保前后一致。
  - 明确‘98%以上’的具体计算基数和考核周期，避免歧义。

## 风险点23：工程技术人员资质要求可能构成倾向性限制

- 问题定性：中风险
- 审查类型：人员资质倾向性审查
- 原文位置：证据6，第716-721行
- 原文摘录：至少有1人具有...特种设备安全管理和作业人员证...或...特种设备安全管理人员证...该人员同时持有有效的...高压电工作业证的得1分...该人员同时还持有有效的...低压电工作业证的得1分
- 风险判断：
  - 评分标准要求同一人员同时持有特种设备证、高压电工证和低压电工证。
  - 此类‘三证合一’的要求在物业管理服务中可能过于严苛，导致具备单一专业资质的人员无法得分，实质上限制了潜在投标人数量，可能构成以不合理条件限制或排斥潜在供应商。
  - 需核实项目实际服务内容是否确实需要同一人同时具备这三项资质，若仅为评分项而非准入项，建议调整评分逻辑。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（二）设定的资格、技术、商务条件与采购项目的具体特点和实际需要不相适应或者与合同履行无关。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 建议将‘同一人同时持有’改为‘拟派团队中至少有1人持有...，至少有1人持有...’，或根据实际岗位需求拆分评分项。
  - 若确需一人多证，需提供充分的合理性说明，证明该岗位必须同时具备三项技能。

  - 《政府采购促进中小企业发展管理办法》（财库〔2020〕46号）：不得以不合理的条件对中小企业实行差别待遇或者歧视待遇。
- 整改建议：
  - 建议根据项目实际绿化面积和养护难度，调整学历和职称要求，或改为‘具有相关从业经验’等更合理的评分因素。
  - 若保留该要求，需在招标文件中说明其必要性。

## 风险点25：最低工资标准引用需确认时效性

- 问题定性：低风险
- 审查类型：数值阈值时效性审查
- 原文位置：证据5，第2628-2658行
- 原文摘录：人工成本包含员工薪资[不低于福州市现行最低工资标准（每人1960元/月）且不含五险一金]
- 风险判断：
  - 招标文件中直接引用了具体的数值‘1960元/月’作为福州市现行最低工资标准。
  - 若该数值在招标发布时已过期或即将调整，可能导致投标人报价依据错误，或合同履行期间产生争议。
  - 虽然文中提到‘合同履行期间，上述标准有调增...中标人应无条件补足’，但固定数值的引用仍需谨慎。
- 法律/政策依据：
  - 《中华人民共和国劳动法》第四十八条：国家实行最低工资保障制度。最低工资的具体标准由省、自治区、直辖市人民政府规定，报国务院备案。

- 整改建议：
  - 建议将‘1960元/月’修改为‘不低于福州市人民政府发布的现行最低工资标准’，并删除具体数值，或注明‘以投标截止时福州市最新公布的标准为准’。
  - 确保招标文件中的数值与最新政策文件一致。
