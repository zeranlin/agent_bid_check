## 风险点1：评分标准中设置特定品牌倾向性描述

- 问题定性：高风险
- 审查类型：评分因素不相关/技术参数倾向性
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 加分项（F4×A4）
- 原文摘录：“设备配置如果有华为小米等相关国产品牌优先。”
- 风险判断：
  - 该条款出现在“加分项”中，且明确列举了“华为”、“小米”等具体品牌名称。
  - 在政府采购中，除特定情形外，不得指定或者变相指定特定品牌、型号。
  - 该描述具有明显的指向性，排斥了其他同等性能的国产品牌供应商，违反了公平竞争原则。
  - 物业管理服务项目的设备配置通常属于通用设备，不应通过指定品牌进行加分。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品……
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 删除“华为小米等相关国产品牌优先”的表述。
  - 如确需鼓励国产品牌，应表述为“优先采购符合国家标准且为国产自主品牌的产品”，不得列举具体品牌名称。
  - 建议删除该加分项，或改为对设备性能、节能指标等通用技术指标进行评分。

## 风险点2：评分标准中设置与履约能力关联度不高的企业荣誉及特定荣誉加分

- 问题定性：中风险
- 审查类型：评分因素不相关
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 商务项（F3×A3）-> 1.企业荣誉、8.项目荣誉
- 原文摘录：
  - “1.企业荣誉：投标人2020年1月1日以来……获得过地市级（含）以上人民政府或行政机关颁发的与物业服务或企业诚信经营有关的荣誉或表彰的，每提供1份得1分，满分3分。”
  - “8.项目荣誉：……获得国家部委组织的‘节约型公共机构示范单位’或‘节约型机关’创建工作荣誉的得0.5分……获得地市级（含）以上人民政府或行政机关授予‘城区生活垃圾分类工作先进单位’荣誉的得0.5分……获得地市级（含）以上人民政府或行政机关授予‘绿化或园林式单位’荣誉的得0.5分。”
- 风险判断：
  - “企业荣誉”项中“与物业服务或企业诚信经营有关”的表述过于宽泛，缺乏量化标准，且荣誉的获取往往与企业的历史积累、公关能力相关，与本项目具体的履约能力（如人员素质、管理方案、设备配置）无直接必然联系。
  - “项目荣誉”中要求提供“节约型公共机构示范单位”、“城区生活垃圾分类工作先进单位”等特定荣誉，这些荣誉通常是针对特定项目或特定单位（如机关单位）的，要求投标人提供此类荣誉作为加分项，可能限制了大量具备履约能力但未获得此类特定荣誉的中小企业参与竞争。
  - 根据《政府采购货物和服务招标投标管理办法》第五十五条，评审因素应当细化和量化，且与相应的商务条件和采购需求对应。荣誉类加分若不能直接证明履约能力，易被认定为不合理条件。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。商务条件和采购需求指标有区间规定的，评审因素不得超出区间规定的范围。
  - 《关于促进政府采购公平竞争优化营商环境的通知》（财库〔2019〕38号）：不得将供应商的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。（虽未直接禁止荣誉，但强调不得设置与履约无关的条件）。
- 整改建议：
  - 建议删除“企业荣誉”和“项目荣誉”中的特定荣誉加分项。
  - 如确需考察企业信誉，可改为考察“近三年内无重大违法记录”（作为资格条件）或“近三年内无因服务质量问题被投诉或处罚的记录”（需提供相关证明），将重点放在履约表现而非荣誉获取上。
  - 若保留，需论证该荣誉与本项目履约能力的直接相关性，并扩大荣誉范围，避免指向特定类型单位。

## 风险点3：评分标准中人员配置要求存在重复计分及社保缴纳时间限制过严

- 问题定性：中风险
- 审查类型：评分因素不相关/限制竞争
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 技术项（F2×A2）-> 17-24项（项目经理、行政专员、安防主管等配置情况）
- 原文摘录：
  - “注：投标人除按评分要求提供相关证明材料外，还应提供拟派的项目经理在投标截止时间前六个月（不含投标截止时间当月）任意一个月投标人为其缴纳社保的证明材料复印件……否则均不得分。第17-24评分项提供的人员不得重复，提供重复人员不得分。”
  - 各岗位（如项目经理、工程主管等）均要求提供“投标截止时间前六个月（不含投标截止时间当月）任意一个月”的社保缴纳证明。
- 风险判断：
  - **社保时间限制过严**：要求提供“投标截止时间前六个月（不含当月）任意一个月”的社保，对于新成立企业、人员流动正常的企业或季节性用工企业可能构成不合理限制。通常要求提供“近3个月”或“近6个月”的连续社保更为合理，或者允许提供“近6个月内任意连续3个月”的社保。仅要求“任意一个月”虽看似宽松，但结合“不含当月”及“六个月”的时间跨度，可能排除部分刚入职或社保缴纳记录有短暂中断但实际在职的优秀人员。
  - **人员重复限制**：要求“第17-24评分项提供的人员不得重复”，即项目经理、行政专员、安防主管等关键岗位必须由不同人员担任。虽然这符合管理逻辑，但在评分项中强制要求“不得重复”且“提供重复人员不得分”，若招标文件未明确这些岗位在技术需求中必须分设，则可能构成对人员配置的过度限制。需结合第五章技术需求确认是否必须分设。
  - **评分项设置**：将学历、证书等作为加分项是常见的，但需确保这些条件与项目实际管理需求直接相关。例如，要求“绿化工”具有“中级（含）以上园林绿化工程师职称”可能要求过高，普通绿化工通常只需具备相关技能证书即可，职称要求可能限制了大量具备实操能力的中小企业。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素……
  - 《关于促进政府采购公平竞争优化营商环境的通知》（财库〔2019〕38号）：不得设置与采购项目的具体特点和实际需要不相适应的资格、技术、商务条件。
- 整改建议：
  - 调整社保缴纳证明要求，建议改为“提供投标截止时间前近6个月内任意连续3个月的社保缴纳证明”，或“提供近6个月内任意1个月的社保缴纳证明（含当月）”，以体现人员稳定性同时兼顾灵活性。
  - 核实第五章技术需求中是否明确要求各岗位必须由不同人员担任。若未强制要求，建议删除“提供重复人员不得分”的表述，或允许在满足岗位资质前提下由同一人兼任（需符合劳动法规定）。
  - 重新评估“绿化工”等基层岗位的职称要求，建议改为“具备相关职业技能等级证书”或“具有X年以上相关经验”，降低门槛，扩大竞争范围。

## 风险点4：商务条款中费用承担范围过宽，可能导致报价虚高或履约风险

- 问题定性：中风险
- 审查类型：商务条款失衡
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 其他商务要求 -> 2)相关费用约定方面
- 原文摘录：
  - “a.货物及杂物搬运过程中产生的费用由中标人承担。”
  - “b.化粪池、雨污井、阀门井、排污管道、蓄水池、水塔、厨房隔油池因清洗、清掏及疏通以上给排水设施、设备而发生的费用由中标人承担。”
  - “c.院区（包括公用部分、办公室及实验室）所有的灯具、灯管等零星的更换，材料由中标人承担。”
  - “h.外墙根据采购人需求清洗，3米（含）以下的保洁每半年至少一次、3米以上的保洁每一年至少一次，外墙清洗相关费用由中标人承担。”
  - “m.如因中标人管理不善导致花草树木损坏或死亡，绿化的更新或补植由中标人负责，费用由中标人承担。中标人须提供476盆绿植供采购人使用……其费用包含在投标价格中，不再另行增加。”
  - “p.中标人负责为本项目配备1辆移动升降机或登高作业车……2辆液压车，2台高压清洗机，2台鼓风机……所产生的一切费用由中标人负责。”
- 风险判断：
  - **费用包干范围过大**：招标文件要求中标人承担几乎所有零星维修、清洗、设备购置（如升降机、高压清洗机等）、绿植租摆及更换等费用。虽然“包干”是常见做法，但部分费用（如外墙清洗、大型设备购置、大量绿植租摆）金额较大且不可控，若未设定上限或明确标准，可能导致中标人为了中标而压低报价，后期通过降低服务质量或偷工减料来弥补成本，甚至因成本过高而违约。
  - **设备购置责任**：要求中标人“配备”移动升降机、液压车等设备并“常驻院区”，且费用全包。对于物业服务企业而言，购置此类大型设备成本较高，且使用频率可能不高，这可能导致只有大型物业公司才能承担，限制了中小企业的参与。
  - **绿植租摆**：要求提供476盆绿植并负责养护、更换，费用全包。绿植的损耗、季节更换成本较高，若未明确损耗率标准，易引发争议。
- 法律/政策依据：
  - 《中华人民共和国政府采购法》第二十二条：供应商参加政府采购活动应当具备下列条件：……（二）有依法缴纳税收和社会保障资金的良好记录；（三）具有良好的商业信誉和健全的财务会计制度……（隐含要求合同条款公平合理）。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）：采购需求应当符合法律法规以及政府采购政策规定的技术、服务、安全等要求。
- 整改建议：
  - 对大额费用项目（如外墙清洗、大型设备购置、绿植租摆）设定费用上限或明确计价标准，或改为“由采购人另行支付”或“在预算范围内据实结算”。
  - 对于设备配备，可改为“中标人应确保具备相应的作业能力，可自有或租赁设备”，不强制要求“购置”并“常驻”，降低准入门槛。
  - 明确绿植租摆的损耗标准、更换频率及费用承担比例，避免无限责任。
  - 建议将部分大额、不可控费用从“包干”中剥离，单独列项或设定最高限价。

## 风险点5：技术参数中部分岗位人员要求存在性别、年龄等歧视性倾向

- 问题定性：中风险
- 审查类型：资格条件/技术参数倾向性
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （二）物业管理团队人员配备情况及要求 -> 1.两院区人员配备情况
- 原文摘录：
  - “行政专员：女性，身体健康，仪表端庄……身高1.60米以上。”
  - “安防主管：需为男性……”
  - “门岗：需为男性……”
  - “院区巡逻岗：需为男性……”
  - “消监控岗：……政治面貌清白，无纹身，无劣迹，无犯罪前科……”
- 风险判断：
  - **性别限制**：明确要求“行政专员”为“女性”，“安防主管”、“门岗”、“巡逻岗”为“男性”。除国家规定的特殊工种外，政府采购不得以性别为由限制供应商。虽然某些岗位（如安保）传统上男性居多，但法律并未禁止女性从事安保工作，强制要求“需为男性”涉嫌就业歧视，限制了女性就业及提供服务的供应商。
  - **身高限制**：要求“行政专员”身高“1.60米以上”，该条件与履行行政专员职责无直接必然联系，属于不合理的身体条件限制。
  - **年龄限制**：部分岗位设定了严格的年龄范围（如25-55周岁），虽有一定合理性，但需确保该范围与岗位实际体力、精力需求相匹配，避免对大龄或年轻求职者构成歧视。
- 法律/政策依据：
  - 《中华人民共和国就业促进法》第二十七条：国家保障妇女享有与男子平等的劳动权利。用人单位招用人员，除国家规定的不适合妇女的工种或者岗位外，不得以性别为由拒绝录用妇女或者提高对妇女的录用标准。
  - 《中华人民共和国政府采购法》第二十二条：供应商参加政府采购活动应当具备下列条件：……（一）具有独立承担民事责任的能力；（二）具有良好的商业信誉和健全的财务会计制度……（隐含不得设置歧视性条件）。
  - 《关于促进政府采购公平竞争优化营商环境的通知》（财库〔2019〕38号）：不得设置与采购项目的具体特点和实际需要不相适应的资格、技术、商务条件。
- 整改建议：
  - 删除所有关于“男性”、“女性”的性别限制，改为“身体健康，能胜任岗位要求”。
  - 删除“身高1.60米以上”的身高限制，除非能证明该身高是履行行政专员职责的绝对必要条件（通常很难证明）。
  - 重新审视年龄限制，建议放宽至“符合国家法定劳动年龄且身体健康，能胜任岗位要求”，或设定更宽泛的年龄区间。

## 风险点6：履约验收条款中考核标准过于严苛且扣款比例过高

- 问题定性：中风险
- 审查类型：商务条款失衡
- 原文位置：第五章 招标内容及要求 -> （五）质量考核办法 -> 物业管理服务负面清单和扣款一览表
- 原文摘录：
  - “考核基金：每月物业管理费的20%”
  - “1.非采购人原因，中标人存在欠薪行为……视情节扣除考核基金的10%—100%”
  - “2.中标人以任何名义和理由将项目进行转包、分包的。扣除考核基金的100%”
  - “13.中标人未按招标文件要求，存在人员数量缺少和缺岗的情况……未提供本项目的员工工资发放一览表的按每人次扣减物业费人民币10000元”
  - “25.因中标人原因，发生安全事故的……视情节扣除考核基金的10%—100%”
- 风险判断：
  - **考核基金比例过高**：每月预留20%的物业费作为考核基金，比例较高，可能影响中标人的现金流，增加履约风险。
  - **扣款标准模糊且严厉**：如“视情节扣除考核基金的10%—100%”，缺乏具体的量化标准，赋予采购人过大的自由裁量权，易引发争议。
  - **单次扣款金额巨大**：如“未提供员工工资发放一览表的按每人次扣减物业费人民币10000元”，该金额对于单次程序性违规（未提供表格）而言过高，且未区分主观恶意与客观疏忽，可能导致中标人因小错而遭受巨额损失，显失公平。
  - **安全事故扣款**：发生安全事故即扣除10%-100%考核基金，未区分事故等级和责任大小，可能过于严苛。
- 法律/政策依据：
  - 《中华人民共和国民法典》第六条：民事主体从事民事活动，应当遵循公平原则，合理确定各方的权利和义务。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）：采购文件应当明确合同的主要条款，包括验收标准、违约责任等。
- 整改建议：
  - 降低考核基金比例，建议调整为每月物业费的5%-10%。
  - 细化扣款标准，将“视情节”改为具体的量化指标（如：缺岗1人扣X元，缺岗2人扣Y元）。
  - 调整单次扣款金额，使其与违规行为的严重程度相匹配。对于程序性违规（如未提供表格），建议以警告、限期整改为主，或设定较低的扣款额度（如500-1000元/次）。
  - 明确安全事故的分级处理机制，根据事故等级和责任大小设定不同的扣款比例。

## 风险点7：技术参数中部分设备配置要求指向特定品牌或型号

- 问题定性：低风险
- 审查类型：技术参数倾向性
- 原文位置：第五章 招标内容及要求 -> H.其他服务 -> 2.实验服清洗服务
- 原文摘录：“由中标人提供8台全新滚筒洗衣机（容积大于等于10公斤、洗烘一体，如：美的、海尔、小天鹅同品质品牌）供检验科所使用……”
- 风险判断：
  - 虽然使用了“如：美的、海尔、小天鹅同品质品牌”的表述，但列举了三个具体品牌，且要求“同品质”，在实际评审中容易引导评委倾向于这三个品牌，排斥其他同等品质的品牌。
  - 建议改为“容积大于等于10公斤、洗烘一体，品牌不限，但需为市场主流品牌且具备相关质量认证”，避免列举具体品牌。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：……（三）采购需求中的技术、服务等要求指向特定供应商、特定产品……
- 整改建议：
  - 删除“如：美的、海尔、小天鹅”的具体品牌列举。
  - 改为“容积大于等于10公斤、洗烘一体，品牌不限，但需为市场主流品牌且具备相关质量认证（如3C认证）”。

## 风险点8：评分标准中“信息化软件服务能力”要求著作权人为投标人

- 问题定性：低风险
- 审查类型：评分因素不相关/限制竞争
- 原文位置：第四章 资格审查与评标 -> 7.2 评标标准 -> 商务项（F3×A3）-> 6.信息化软件服务能力
- 原文摘录：“投标人拥有与物业服务相关信息化管理软件或系统，并承诺在服务期内该软件或系统会投入本项目使用的，每提供1项得1分，满分3分。投标人须提供承诺函和著作权人为投标人的《计算机软件著作权登记证书》……”
- 风险判断：
  - 要求“著作权人为投标人”可能限制了大量使用成熟第三方软件或购买软件服务的企业参与竞争。物业服务企业通常使用成熟的第三方管理软件，而非自行开发。
  - 该要求与项目履约能力（即能否有效管理物业）无直接必然联系，重点应在于软件的功能、稳定性及与项目的匹配度，而非著作权归属。
- 法律/政策依据：
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。
- 整改建议：
  - 修改为“投标人拥有或合法使用与物业服务相关信息化管理软件或系统……”，允许提供软件著作权证书或合法授权书。
  - 重点考察软件的功能模块、数据安全性、易用性等与项目需求相关的指标。
