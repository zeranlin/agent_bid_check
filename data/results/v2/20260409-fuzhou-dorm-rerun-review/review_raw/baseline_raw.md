## 风险点1：技术参数引用错误标准及标准号存疑

- 问题定性：高风险
- 审查类型：技术参数倾向性/合规性
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> 序号1：组合柜 -> 第8点；序号2：学生宿舍座椅 -> 第8点；序号3：楼梯踏板 -> 第4点
- 原文摘录：
  - 组合柜第8点：“家具中甲醛、苯、二甲苯和TVOC的释放量以及家具中可迁移有害元素限量需要符合GB 28481-2012《家具中有害物质限量》标准以及GB 48001-2026《汽车车门把手安全技术要求》标准中的要求。”
  - 座椅第8点：“家具中甲醛、苯、二甲苯和TVOC的释放量，家具中可迁移有害元素限量，以及领苯二甲酸脂、多环芳烃限量需要符合GB18584-2024《家具中有的害物质限量测量》标准中的要求。”
  - 楼梯踏板第4点：“家具中可迁移有害元素限量，以及领苯二甲酸脂、多环芳烃限量需要符合GB18584-2024《家具中有害物质限量》标准中的要求并提供福建省检测机构出具的检测报告。”
- 风险判断：
  - **标准引用严重错误**：组合柜要求中引用了"GB 48001-2026《汽车车门把手安全技术要求》”，该标准名称与家具采购标的完全无关，且年份为2026年（未来时间），属于明显的编造或错误引用，导致技术参数无法执行，构成排斥潜在供应商。
  - **标准号及名称存疑**：座椅和楼梯踏板引用的"GB18584-2024"及《家具中有的害物质限量测量》（错别字“有的害”）存疑。现行有效标准通常为GB 18584-2001《室内装饰装修材料 木家具中有害物质限量》或GB 28481-2012《家具中有害物质限量》。2024版标准是否存在需核实，且标准名称存在明显文字错误，可能导致评审依据混乱。
  - **地域限制风险**：楼梯踏板要求“提供福建省检测机构出具的检测报告”，限制了外地检测机构出具的报告，涉嫌以不合理的条件限制或者排斥潜在供应商。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（三）采购需求中的技术、服务等要求指向特定供应商、特定产品；（八）以其他不合理条件限制或者排斥潜在供应商。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第十七条：采购人、采购代理机构不得将投标人的注册资本、资产总额、营业收入、从业人员、利润、纳税额等规模条件作为资格要求或者评审因素，也不得通过将除进口货物以外的生产厂家授权、承诺、证明、背书等作为资格要求，对投标人实行差别待遇或者歧视待遇。
- 整改建议：
  - 立即核实并修正所有引用的国家标准号及标准名称，删除与家具无关的“汽车车门把手”标准。
  - 确认GB 18584的最新版本及正确名称，修正错别字。
  - 删除“福建省检测机构”的地域限制，改为“具有CMA或CNAS资质的检测机构出具的检测报告”。

## 风险点2：样品制作要求具有排他性及泄露信息风险

- 问题定性：高风险
- 审查类型：技术参数倾向性/限制竞争
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> （二）样品要求 -> 第3点、第4点
- 原文摘录：
  - 第3点：“投标人应采用可去除的安全材料隐去投标人名称及其他可能泄露投标人信息的标志、符号等；同时，在投标文件中，也不得出现样品图样，标识等可能泄露投标人样品的任何信息，否则按无效投标处理。”
  - 第4点：“投标人必须将以上样品成品提前组装好，并于开标当日投标截止时间前一次性送达投标现场，按要求摆放好样品。必要时评标现场评委可对所有样品进行测试，若出现复核情况时，评标委员会对样品不作再次评审，样品评审结果不作更改。”
- 风险判断：
  - **样品评审方式风险**：要求“开标当日投标截止时间前一次性送达投标现场”，对于大型家具（如组合柜、楼梯踏板）的样品，运输和现场组装难度大，且“提前组装好”可能暗示对特定生产工艺的依赖。
  - **信息泄露与无效投标风险**：虽然要求隐去信息，但要求“不得出现样品图样”，而招标文件第五章中包含了大量详细的“参考图”和“样品制作要求”附图。如果投标人的样品与这些附图高度一致（因为必须响应参数），评委极易通过样品的独特设计（如特定的防撞角形状、拉手内凹设计、抽屉盒加强筋数量等）识别出投标人，导致“隐去信息”的要求形同虚设，反而可能因无法完全隐去而被判定无效，构成对不熟悉该特定设计图纸的供应商的排斥。
  - **参数指向性**：样品要求中详细规定了“防撞角外形以及规格尺寸见附图”、“拉手主体和拉手盖板的外形以及规格尺寸见附图”等，这些高度定制化的外观要求，极可能指向特定供应商的现有产品，限制了其他供应商参与。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（三）采购需求中的技术、服务等要求指向特定供应商、特定产品。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条：评审因素应当细化和量化，且与相应的商务条件和采购需求对应。
- 整改建议：
  - 重新评估样品要求的必要性。对于家具类项目，通常可通过提供关键部件样品或详细的技术参数响应表来评审，而非要求全套成品样品。
  - 若必须提供样品，应允许提供关键部件（如门板、抽屉盒、座椅背板等）而非整柜，降低运输和组装难度。
  - 删除“不得出现样品图样”的绝对化表述，或明确允许在投标文件中提供与样品一致的图纸作为技术响应，避免因图纸与样品一致而被误判为泄露信息。
  - 审查附图中的设计是否过于独特，若存在指向特定品牌/型号的风险，应修改为功能性描述而非外观描述。

## 风险点3：商务条款中采购人单方变更权过大且结算方式不明

- 问题定性：中风险
- 审查类型：商务条款失衡
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 其他商务要求 -> 8.3、13.3
- 原文摘录：
  - 8.3：“签订合同后，采购单位有权根据实际需要对本次采购文件中未明确或未规定的事项作出规定或细微调整，及对各种货物的款式进行细微调整（允许偏离±5%），最终以采购人实际为准，中标人的最终报价不变，如遇采购人特殊需求特殊制作与招标尺寸有较大差异的货物，偏离超过5%，结算单价按调整后面积比例相应换算。”
  - 13.3：“①本项目中标后，采购人有权对家具的颜色、款式、尺寸等作出细微调整，中标价不变，中标人不得因此增加任何费用或持有任何异议。②...采购人根据现场的实际情况，有权对所有货物的数量、尺寸和款式进行增加、减少或调整，最终以采购人实际的安装数量为准，变化部分的单价与投标报价对应的单价一致..."
- 风险判断：
  - **权利义务不对等**：条款赋予采购人极大的单方变更权（颜色、款式、尺寸、数量），且规定“中标价不变”、“不得增加任何费用”。家具定制涉及材料成本、工艺成本，若款式或尺寸调整导致成本增加（如从普通板材改为特殊工艺，或尺寸变化导致材料利用率下降），强制要求中标人承担所有成本风险，显失公平。
  - **结算依据模糊**：“按调整后面积比例相应换算”缺乏明确的计算公式和单价调整机制。家具通常按“套”或“件”计价，而非单纯按“面积”换算，该表述可能导致结算争议。
  - **排斥竞争**：潜在供应商在报价时无法准确预估变更风险，可能导致报价虚高或中标后无法履约，实质上限制了愿意承担合理风险但无法接受无限责任的供应商参与。
- 法律/政策依据：
  - 《中华人民共和国民法典》第五百四十三条：当事人协商一致，可以变更合同。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第三十六条：采购人或者采购代理机构不得在招标文件中规定中标人必须接受采购人提出的不合理要求。
- 整改建议：
  - 修改变更条款，明确变更需经双方协商一致，并约定变更导致成本增减时的价格调整机制（如：变更幅度在X%以内不调整价格，超过X%部分按约定单价或市场价调整）。
  - 明确“面积比例换算”的具体定义，或改为按“实际安装数量×中标单价”结算，并约定特殊工艺变更的计价原则。
  - 删除“不得持有任何异议”等霸王条款。

## 风险点4：验收条款中驻厂检查及终止合同条件过于严苛

- 问题定性：中风险
- 审查类型：商务条款失衡/验收条款
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 其他商务要求 -> 10.4
- 原文摘录：
  - 10.4：“投标人须承诺：若中标，在生产本次采购家具的过程中，采购人可对产品从进料、开胚、酸洗磷化、喷涂、成型等一系列生产程序进行驻厂检查、监督；如果采购人在检查监督过程中发现中标人未按照相关工艺要求进行生产，采购人有权终止合同，并按照国家相关法律法规的规定提请有关政府监管部门对其进行处理。”
- 风险判断：
  - **驻厂检查范围过宽**：要求对“进料、开胚、酸洗磷化、喷涂、成型”等全工序进行驻厂检查，对于家具制造企业而言，这将极大干扰正常生产秩序，且采购人通常不具备专业驻厂监督能力。
  - **终止合同条件过于严苛**：仅因“发现未按照相关工艺要求生产”即可直接“终止合同”，未区分一般瑕疵与重大违约，也未给予整改机会。这可能导致因微小工艺偏差（如非关键部位的喷涂厚度略微不足）而直接废标，风险与后果不成比例。
  - **涉嫌排斥**：部分中小企业可能因生产场地限制或管理流程原因，无法配合如此高频、深度的驻厂检查，从而被变相排斥。
- 法律/政策依据：
  - 《中华人民共和国政府采购法》第四十三条：政府采购合同适用合同法。采购人和供应商之间的权利和义务，应当按照平等、自愿的原则以合同方式约定。
  - 《政府采购货物和服务招标投标管理办法》（财政部令第87号）相关精神：合同条款应公平合理，不得设置不合理的违约责任。
- 整改建议：
  - 将“驻厂检查”改为“不定期抽查”或“关键节点验收”，明确检查的频率和范围。
  - 修改终止合同条件，增加“经采购人提出整改要求后，中标人未在规定期限内整改或整改后仍不符合要求”的前置程序，区分一般违约与根本违约。

## 风险点5：履约保证金退还时限与法律规定不一致

- 问题定性：低风险
- 审查类型：商务条款合规性
- 原文位置：第五章 招标内容及要求 -> 三、商务要求 -> 6.★合同支付方式 -> 7.★履约保证金
- 原文摘录：
  - “该履约保证金将在验收合格后且中标人无违约的前提下10个工作日内无息退还（转账方式提交的）或解除担保（保函形式提交的）。”
- 风险判断：
  - 虽然10个工作日在合理范围内，但需确认是否与《政府采购法实施条例》及财政部关于优化营商环境的相关文件（如《关于促进政府采购公平竞争优化营商环境的通知》财库〔2019〕38号）中关于“及时退还”的要求完全契合。通常建议明确为“验收合格后X个工作日内”，10个工作日尚可，但需确保实际操作中不拖延。
  - 此处风险较低，主要作为提示，确保执行时严格遵守时限。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第四十八条：采购文件要求中标人提交履约保证金的，中标人应当按照合同约定提交。履约保证金不得超过政府采购合同金额的10%。
  - 《关于促进政府采购公平竞争优化营商环境的通知》（财库〔2019〕38号）：不得违规收取保证金，不得拖延退还保证金。
- 整改建议：
  - 保持现状，但在合同执行中严格监控退还时限，确保不超过10个工作日。

## 风险点6：技术参数中部分指标描述模糊或存在逻辑矛盾

- 问题定性：中风险
- 审查类型：技术参数倾向性/评分因素不相关
- 原文位置：第五章 招标内容及要求 -> 二、技术和服务要求 -> 序号1：组合柜 -> 第1点、第7点
- 原文摘录：
  - 第1点：“档条底部需要跟桌面采用三合一连接件固定，背面需要跟装饰板采用螺丝固定，不能出现松动和异响。”
  - 第7点：“抽屉盒底部具有≥63个镂空条形孔，条形孔的宽度＜5mm，避免笔等物品掉落。”
- 风险判断：
  - **指标过于具体**：“≥63个镂空条形孔”、“条形孔宽度＜5mm”等参数过于具体，且与家具的核心功能（承重、耐用、环保）关联度不高，极可能是特定厂家产品的独有特征，具有明显的指向性。
  - **逻辑矛盾**：要求“避免笔等物品掉落”，但条形孔宽度＜5mm，对于某些细长物品可能仍会掉落，且“63个”这一具体数字缺乏科学依据，属于典型的“控标”参数。
- 法律/政策依据：
  - 《中华人民共和国政府采购法实施条例》第二十条：采购人或者采购代理机构有下列情形之一的，属于以不合理的条件对供应商实行差别待遇或者歧视待遇：（三）采购需求中的技术、服务等要求指向特定供应商、特定产品。
- 整改建议：
  - 将“≥63个”改为“具有足够的镂空条形孔以满足通风/排水需求”或设定合理的范围（如“不少于50个”），删除过于精确的数字。
  - 将“条形孔宽度＜5mm”改为“条形孔宽度应小于常见文具直径（如5mm）”或设定合理范围，避免指向特定模具。

## 综合判断

- 高风险问题：
  - 风险点1：技术参数引用错误标准及标准号存疑（引用汽车标准、未来标准、地域限制）。
  - 风险点2：样品制作要求具有排他性及泄露信息风险（高度定制化外观、样品送达要求）。
- 中风险问题：
  - 风险点3：商务条款中采购人单方变更权过大且结算方式不明。
  - 风险点4：验收条款中驻厂检查及终止合同条件过于严苛。
  - 风险点6：技术参数中部分指标描述模糊或存在逻辑矛盾（具体孔数、宽度）。
- 需人工复核事项：
  - 需核实GB 18584-2024标准是否真实存在及正确名称。
  - 需核实“福建省检测机构”要求是否违反最新优化营商环境政策（虽已判定为高风险，但需确认是否有特殊政策依据）。
