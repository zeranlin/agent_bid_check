# R-011 资质证明材料原件及纸质证照要求边界识别

- 任务编号：R-011
- 状态：开发中
- 目标标题：要求提供资质证照原件或电子证照纸质件，存在材料提交边界设置不当风险

## 任务目标

补齐“资质证明材料原件及纸质证照要求边界”识别，让系统能够稳定识别：

1. 资格材料、资格证明文件、证照提交要求中出现原件、纸质证照、纸质版电子证照等表达。
2. 上述要求被明确设置为必须提交的资格或响应材料。
3. 最终输出标准化正式风险。

## 本轮范围

1. 补材料提交要求侧信号提取。
2. 补原件 / 纸质证照风险语义提取。
3. 补投标阶段门槛判断与 cross-check。
4. 补正样本、负样本、规则注册和端到端测试。

## 核心边界

1. 不误报中标后核验原件场景。
2. 不误报法定明确要求的核验场景。
3. 不把扫描件、电子签章件、电子证照查询方式误判为正式风险。
4. 输出口径必须对准投标阶段材料提交边界风险。
