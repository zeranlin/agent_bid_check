# R-012 供应商主体身份与地域限制识别

- 任务编号：R-012
- 状态：已通过
- 目标标题：以供应商主体身份或地域条件设置准入门槛，存在限制竞争风险

## 任务目标

补齐“供应商主体身份与地域限制”识别，让系统能够稳定识别：

1. 资格条件、合格供应商条件、资格审查条款中出现主体身份或地域要求。
2. 条款语义命中所有制形式、组织形式、注册地、所在地、分支机构、经营网点等表达。
3. 相关要求被明确设置为投标准入门槛、合格供应商条件或资格审查依据。
4. 最终输出标准化正式风险。

## 本轮范围

1. 补主体身份 / 地域要求侧信号提取。
2. 补主体身份 / 地域限制语义提取。
3. 补准入门槛判断与 cross-check。
4. 补正样本、负样本、规则注册和端到端测试。

## 核心边界

1. 不误报中标后本地服务响应安排。
2. 不误报法定明确地域设置场景。
3. 不把一般履约便利性说明误判为正式风险。
4. 输出口径必须对准准入门槛限制竞争风险。
