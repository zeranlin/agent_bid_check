# R-005 整改任务单

## 基本信息

- 任务编号：`R-005`
- 任务名称：补齐“不得要求提供赠品、回扣或者与采购无关的其他商品、服务”评分合规风险识别
- 任务状态：`待下发`
- 下发对象：`T`
- 监督角色：`M`

## 任务背景

新增评审规则要求：

`评审规则合规性-不得要求提供赠品、回扣或者与采购无关的其他商品、服务`

但当前文件中存在如下评分描述：

`（一）评分内容 投标人针对本项目提供售后服务承诺：承诺在接到采购人通知后，能够在1小时（60分钟）内到达现场处理问题，并额外向采购人值班室赠送台式电脑、打印机各1套的，得100分；仅承诺1.5小时（90分钟）内到达现场处理问题的，得50分；其他情况不得分。（二）评审标准：要求投标人提供承诺（格式自拟）作为得分依据。未提供承诺或承诺内容不满足要求均不得分。`

上述内容已将：

- `额外赠送台式电脑、打印机各1套`

直接作为高分条件，并与 `100分 / 50分 / 不得分` 直接挂钩，存在明显评分规则合规风险。系统目前未稳定识别该问题。

## 整改目标

让系统能够稳定识别：

- 规则明确要求“不得要求提供赠品、回扣或者与采购无关的其他商品、服务”
- 评分项中实际出现赠品、额外商品、额外服务、回扣等内容
- 且这些内容与得分、加分、评审标准直接挂钩
- 最终输出标准风险提示

## 标准风险文案

### 风险标题

`将赠送额外商品作为评分条件，违反评审规则合规性要求`

### 问题定性

`高风险`

### 审查类型

`评分因素合规性 / 不当附加交易条件`

### 风险判断要求

- 评审规则已明确，不得要求提供赠品、回扣或者与采购无关的其他商品、服务。
- 当前评分内容将“额外赠送台式电脑、打印机各1套”作为高分条件。
- 上述赠送内容明显超出采购标的本身，属于与采购无关的额外商品。
- 该类设置容易诱导以额外利益换取评分优势，存在明显不合规风险，并可能导致评审结果失真、供应商公平竞争受损。

### 整改建议要求

- 删除“赠送台式电脑、打印机”等与采购无关的附加商品要求。
- 售后服务评分如需保留，应仅围绕响应时效、服务能力、保障机制等与采购标的直接相关内容设置。
- 对评分标准重新调整，避免将赠品、返利、回扣或无关服务作为加分条件。

## 规则整改要求

### 1. 规则侧信号提取

识别以下语义：

- `不得要求提供赠品`
- `不得要求提供回扣`
- `不得要求提供与采购无关的其他商品`
- `不得要求提供与采购无关的其他服务`
- `赠品不得作为评审因素`
- `回扣不得纳入评分`

建议形成结构化字段，例如：

- `gifts_or_unrelated_goods_forbidden_in_scoring = true`

### 2. 评分项侧信号提取

识别评分内容中的以下表达：

- `赠送`
- `赠品`
- `台式电脑`
- `打印机`
- `额外提供`
- `额外赠送`
- `回扣`
- `返利`
- `无关商品`
- `无关服务`

并判断其是否出现在：

- `评分内容`
- `评审标准`
- `评分因素`
- `得分`
- `加分`
- `得100分`
- `不得分`

建议形成结构化字段，例如：

- `scoring_contains_gifts_or_unrelated_goods = true`
- `gifts_or_goods_scoring_sections = [...]`
- `gifts_or_goods_scoring_sentences = [...]`
- `gifts_or_goods_linked_to_score = true`

### 3. Cross-check 规则

当同时满足：

- 规则要求不得提供赠品、回扣或与采购无关的其他商品、服务
- 评分项中出现此类内容
- 且与得分、加分、分值直接挂钩

则输出该风险点。

## 样本整改要求

### 正样本

至少包括：

- `额外赠送台式电脑、打印机各1套的，得100分`
- `提供赠品的加分`
- `提供回扣或返利承诺的加分`
- `提供与采购无关的额外商品/服务的加分`
- `类似项目业绩中包含值班室办公设备配置或会议保障等综合服务内容的，每提供1个业绩得20分`

### 负样本

至少包括：

- 售后服务评分只涉及响应时效，不涉及赠品
- 评分项只评服务能力、实施方案，不含额外商品
- 文中出现“电脑”“打印机”但属于采购标的本身，不应误报

### 真实业务表达样本

必须补入这条真实写法：

`承诺在接到采购人通知后，能够在1小时（60分钟）内到达现场处理问题，并额外向采购人值班室赠送台式电脑、打印机各1套的，得100分；仅承诺1.5小时（90分钟）内到达现场处理问题的，得50分；其他情况不得分。`

并补入以下隐性扩展样本：

`（一）评分内容：1、提供投标人2022年4月1日至投标截止日承担过类似项目的销售安装业绩，且在项目履约中包含值班室办公设备配置或会议保障等综合服务内容的，每提供1个业绩得20分，其它情况不得分，本项最高得100分。`

该样本用于覆盖：

- 未直接写“赠送”字样
- 但将与本次采购无直接关联的额外商品或综合服务能力作为评分加分条件
- 属于 `R-005` 的隐性高风险变体

## M 的验收标准

### 通过线

- 能稳定识别“赠品 / 回扣 / 无关商品服务进入评分因素”的问题
- 不把正常采购标的或正常售后承诺误报成违规
- 风险文案完整、可直接给业务使用
- 回归测试齐全

### 不过线情形

- 只识别“赠送”字样，不判断是否进入评分
- 只识别评分，不对齐规则禁令
- 误把采购标的本身的商品配置报成赠品违规
- 没有真实业务表达样本和端到端测试
