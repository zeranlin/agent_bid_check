# R-006 整改任务单

## 基本信息

- 任务编号：`R-006`
- 任务名称：补齐“不得限定或者指定特定的专利、商标、品牌或者供应商”评分合规风险识别
- 任务状态：`待下发`
- 下发对象：`T`
- 监督角色：`M`

## 任务背景

新增评审规则要求：

`评审规则合理性-不得限定或者指定特定的专利、商标、品牌或者供应商`

但当前文件中存在如下评分描述：

`（一）评分内容：柴油发电机组制造商的投标的柴油发电机组，具备以下认证的：1.具备有效期内省级标准协会颁发的省级采用国际标准产品确认证书和采用国际标准产品标志证书的，每具备一项得40分，具备两项得80分，本小项最高得80分。2.具备CNAS中国认可产品标志证书的，得20分，本小项最高得20分。以上累计最高得分为100分。（二）证明文件：须提供有效期内的认证证书扫描件，证明材料未提供或未按要求提供或不清晰不得分。`

上述内容已将：

- `柴油发电机组制造商`
- `省级采用国际标准产品确认证书`
- `采用国际标准产品标志证书`
- `CNAS中国认可产品标志证书`

直接作为高分条件，并与 `每具备一项得40分 / 得20分 / 最高得100分` 直接挂钩，存在明显评分规则合规风险。系统目前未稳定识别该问题。

## 整改目标

让系统能够稳定识别：

- 规则明确要求“不得限定或者指定特定的专利、商标、品牌或者供应商”
- 评分项中实际出现制造商限定、指定认证证书、指定认证体系、指定标志证书等内容
- 且这些内容与得分、加分、评审标准直接挂钩
- 最终输出标准风险提示

## 标准风险文案

### 风险标题

`以制造商特定认证证书作为高分条件，存在限定特定供应商和倾向性评分风险`

### 问题定性

`高风险`

### 审查类型

`评分因素合规性 / 限定特定供应商或认证体系`

### 风险判断要求

- 评审规则已明确，不得限定或者指定特定的专利、商标、品牌或者供应商。
- 当前评分内容将制造商的特定认证证书、特定标志证书作为高分条件。
- 上述证书要求明显偏向少数具备特定认证体系的供应商或制造商，容易形成倾向性评分。
- 该类设置可能限制公平竞争，并导致评审结果失真或争议增大。

### 整改建议要求

- 删除以特定认证体系、特定标志证书作为高分条件的评分设置。
- 如确需评价产品质量或能力，应改为与采购需求直接相关、可由不同供应商公平满足的通用能力指标。
- 避免通过制造商身份、指定认证来源或指定标志证书形成事实上的供应商限定。

## 规则整改要求

### 1. 规则侧信号提取

识别以下语义：

- `不得限定或者指定特定的专利`
- `不得限定或者指定特定的商标`
- `不得限定或者指定特定的品牌`
- `不得限定或者指定特定的供应商`
- `不得以特定认证作为倾向性评分条件`

建议形成结构化字段，例如：

- `specific_brand_or_supplier_forbidden_in_scoring = true`

### 2. 评分项侧信号提取

识别评分内容中的以下表达：

- `制造商`
- `特定认证证书`
- `标志证书`
- `采用国际标准产品确认证书`
- `采用国际标准产品标志证书`
- `CNAS中国认可产品标志证书`
- `每具备一项得分`
- `累计最高得分`

并判断其是否出现在：

- `评分内容`
- `评分标准`
- `证明文件`
- `得分`
- `加分`
- `最高得分`

建议形成结构化字段，例如：

- `scoring_contains_specific_cert_or_supplier_signal = true`
- `specific_cert_or_supplier_score_linked = true`
- `specific_cert_or_supplier_evidence = [...]`

### 3. Cross-check 规则

当同时满足：

- 规则要求不得限定或者指定特定的专利、商标、品牌或者供应商
- 评分项中出现制造商限定、特定认证证书、指定认证体系或指定标志证书
- 且与得分、加分、分值直接挂钩

则输出该风险点。

## 样本整改要求

### 正样本

至少包括：

- `具备省级采用国际标准产品确认证书的得分`
- `具备采用国际标准产品标志证书的得分`
- `具备CNAS中国认可产品标志证书的得分`
- `制造商具备指定认证的得高分`
- `指定品牌/指定供应商/指定认证体系进入评分`

### 负样本

至少包括：

- 普通质量管理能力描述，但不指定证书来源或特定体系
- 对技术参数、履约能力进行通用描述，不与指定认证绑定
- 仅要求提供合格证明或法定必备资质，不作为倾向性高分条件

### 真实业务表达样本

必须补入这条真实写法：

`柴油发电机组制造商的投标的柴油发电机组，具备以下认证的：1.具备有效期内省级标准协会颁发的省级采用国际标准产品确认证书和采用国际标准产品标志证书的，每具备一项得40分，具备两项得80分，本小项最高得80分。2.具备CNAS中国认可产品标志证书的，得20分，本小项最高得20分。以上累计最高得分为100分。`

## M 的验收标准

### 通过线

- 能稳定识别“指定认证体系 / 指定证书 / 制造商限定进入评分”的问题
- 不把法定资质、通用合格证明误报成违规
- 风险文案完整、可直接给业务使用
- 回归测试齐全

### 不过线情形

- 只识别“证书”字样，不判断是否进入评分
- 只识别评分，不对齐规则禁令
- 误把一般资质要求或合规证明报成特定供应商限定
- 没有真实业务表达样本和端到端测试
