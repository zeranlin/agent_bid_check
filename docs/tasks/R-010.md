# R-010 已取消资质资格认证禁入评分识别

- 任务编号：R-010
- 状态：开发中
- 目标标题：将已取消或非强制资质资格认证作为评审因素，存在评分设置不合规风险

## 任务目标

补齐“已取消资质资格认证禁入评分”识别，让系统能够稳定识别：

1. 评分内容、加分项、评审标准、打分档次中出现资质、资格、认证要求。
2. 条款语义命中“已取消资质 / 已取消资格 / 已取消认证 / 已明令取消 / 非强制认证 / 非强制资格”等表达。
3. 相关要求与得分、加分、分值或档次评价直接挂钩。
4. 最终输出标准化正式风险。

## 本轮范围

1. 补评分因素侧信号提取。
2. 补已取消 / 非强制资质资格认证语义提取。
3. 补评分挂钩判断与 cross-check。
4. 补正样本、负样本、规则注册和端到端测试。

## 核心边界

1. 不误报仅出现在资格条件、未进入评分的条款。
2. 不误报现行法定强制许可类要求。
3. 不把政策背景说明误判为正式风险。
4. 输出口径必须对准评分因素合规风险，不泛化成一般资质问题。

## 验收关注点

1. `scoring_requirement_present = true`
2. `cancelled_or_non_mandatory_scoring_credential_signal = true`
3. `cancelled_or_non_mandatory_scoring_credential_linked_to_score = true`
4. compare 输出标题稳定为：`将已取消或非强制资质资格认证作为评审因素，存在评分设置不合规风险`
5. 正负样本、回放样本和规则注册文件互相对齐
