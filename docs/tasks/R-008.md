# R-008 整改任务单

- 任务编号：`R-008`
- 任务名称：补齐“评分项中要求赠送非项目物资”显性风险识别
- 任务类型：漏报整改 / 规则补强
- 当前状态：待下发
- 提出方：M
- 执行方：T

## 一、问题背景

在真实文件 `[SZDL2025000495-A]柴油发电机组及相关配套机电设备采购及安装项目.docx` 的审查过程中，发现如下评分条款：

`投标人针对本项目提供售后服务承诺：承诺在接到采购人通知后，能够在1小时（60分钟）内到达现场处理问题，并额外向采购人值班室赠送台式电脑、打印机各1套的，得100分；仅承诺1.5小时（90分钟）内到达现场处理问题的，得50分；其他情况不得分。`

该条款已明显触发以下规则方向：

- 不得要求提供赠品、回扣或者与采购无关的其他商品、服务
- 在评分项中要求赠送非项目物资，属于显性风险

但当前系统仅识别出了“售后响应时间”问题，未识别“赠送非项目物资”风险，判定为明确漏报。

## 二、整改目标

补齐系统对“评分项 / 加分项 / 承诺项中要求赠送与采购无关物资”的识别能力，使其能够：

1. 在规则层正确命中此类问题
2. 在输出层明确生成“赠送非项目物资”的正式风险
3. 在真实文件回放中稳定检出，不再被“响应时间类风险”覆盖

## 三、规则定义

- 规则编号：`R-008`
- 规则名称：不得在评分项中要求提供赠品或与采购无关的其他商品、服务

### 触发条件

满足以下条件之一时触发：

- 评分项、加分项、承诺项中出现 `赠送`、`附送`、`额外提供`、`无偿提供` 等表述
- 赠送对象为非采购标的、非必要配套、非清单内组成部分
- 赠送行为与得分、加分、档次评价或中标倾向直接挂钩

### 高风险关键词示例

- `赠送台式电脑`
- `赠送打印机`
- `赠送办公设备`
- `赠送值班室物资`
- `额外提供礼品`
- `附送会议保障用品`

### 负样本 / 排除条件

以下情况不得误报：

- 所谓“赠送”内容本身就是采购标的组成部分
- 属于项目必要配套设备，且已在需求清单、技术参数或报价范围内明确列项
- 属于正常随机附件、安装附件、调试附件
- 与项目履约直接相关的必要服务，不属于额外赠品

## 四、标准输出要求

### 标准风险标题

- 评分项中要求赠送非项目物资，存在明显不当加分和评审合规风险

### 标准风险文案

- 评分条款将向采购人额外赠送与采购标的无关的物资作为加分条件，实质上是以赠品或附加利益影响评审结果。
- 该类设置不属于围绕采购标的履约能力、技术响应或服务能力的正当评分因素，存在明显评审规则合规风险，并可能对供应商形成不当引导。

### 标准整改建议

- 删除赠送台式电脑、打印机等与项目采购无关物资的加分条件。
- 评分因素应仅围绕采购标的履约能力、技术响应、实施组织和服务保障能力设置。
- 不得通过赠品、附加物资或与采购无关的其他商品、服务影响评审结果。

### 应落层级

- `正式风险`

## 五、开发要求

T 需完成以下内容：

1. 在规则库中新增 `R-008`
2. 补充识别模式：
   - `赠送 + 非项目物资 + 得分/加分`
   - `额外提供 + 非采购标的 + 评分挂钩`
3. 补充样本集：
   - 正样本
   - 负样本
   - 混淆样本
4. 补充自动化测试：
   - 规则命中测试
   - 输出层标题/文案测试
   - 真实文件回放测试
5. 确保输出层不要再被“售后响应时间风险”单独覆盖
6. 若同一条款同时存在“响应时间”与“赠品”问题，应优先保留“赠送非项目物资”作为主风险之一

## 六、样本补充要求

### 正样本

- 1小时到场并赠送电脑、打印机得高分
- 提供免费办公家具、值班室设备得加分
- 承诺附送礼品或非项目物资作为评分依据

### 负样本

- 提供采购清单内设备本体及必要附件
- 提供安装辅材、调试辅材、随机配件
- 提供项目必要培训、维保、质保延长等项目相关服务

### 混淆样本

- 赠送耗材但耗材是否属于项目必要配套
- 赠送延保服务但需判断是否与采购标的直接相关
- 赠送培训服务但需判断是否属于正常履约范围

## 七、验收标准

M 验收时按以下标准判定：

1. 真实文件回放必须稳定命中 `R-008`
2. 输出结果中必须出现“赠送非项目物资”风险
3. 风险标题、风险文案、整改建议必须准确
4. 不接受仅输出“响应时间过严”而遗漏赠品风险
5. 不得将采购标的组成部分、必要配套、清单内项目误报为赠品风险
